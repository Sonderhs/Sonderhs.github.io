  • дёҖеҜ№дёҖпјҲдҫӢеҰӮеӣҫеғҸеҲҶзұ»пјүпјҡдёҖдёӘиҫ“е…ҘеҜ№еә”дёҖдёӘиҫ“еҮәгҖӮ

  • еӨҡеҜ№дёҖпјҲдҫӢеҰӮжғ…ж„ҹеҲҶжһҗпјүпјҡеӨҡдёӘиҫ“е…ҘеҜ№еә”дёҖдёӘиҫ“еҮәгҖӮ

1.1.1 еҚ•еҗ‘RNNз»“жһ„

еҚ•еҗ‘RNNзҡ„з»“жһ„еҰӮеӣҫжүҖзӨәпјҡ

RNNзҡ„и®Ўз®—иҝҮзЁӢеҰӮдёӢпјҡ

  1. жҲ‘д»¬еҲқе§ӢеҢ–дёҖдёӘйҡҗи—ҸзҠ¶жҖҒзҹ©йҳөh0h_0h0вҖӢпјҢз„¶еҗҺиҫ“е…Ҙx1x_1x1вҖӢгҖӮ
  2. йҰ–е…ҲпјҢh0h_0h0вҖӢе’Ңx1x_1x1вҖӢеҲҶеҲ«з»ҸиҝҮдёҖдёӘзәҝжҖ§еұӮеҫ—еҲ°h1h_1h1вҖӢпјҢз„¶еҗҺжҲ‘д»¬е°ұеҸҜд»Ҙж №жҚ®h1h_1h1вҖӢеҫ—еҲ°з¬¬дёҖдёӘиҫ“еҮәy1y_1y1вҖӢгҖӮ
  3. д№ӢеҗҺжҲ‘д»¬е°Ҷh1h_1h1вҖӢе’Ңx2x_2x2вҖӢеҶҚеҲҶеҲ«з»ҸиҝҮдёҖдёӘзәҝжҖ§еұӮи®Ўз®—еҫ—еҲ°h2h_2h2вҖӢпјҢз„¶еҗҺжҲ‘д»¬е°ұеҸҜд»ҘеҶҚж №жҚ®h2h_2h2вҖӢи®Ўз®—еҫ—еҲ°y2y_2y2вҖӢгҖӮ
  4. д»ҘжӯӨзұ»жҺЁ

1.1.2 еҚ•еҗ‘еҫӘзҺҜзҪ‘з»ңеңЁPytorchдёӯзҡ„е®һзҺ°

еңЁtorch.nnдёӯпјҢRNNзҡ„е®һзҺ°еҺҹзҗҶеҰӮдёӢпјҡ

ht=tanh(Wihxt+bih+WhhhtвҲ’1+bhh)h_t = tanh(W_{ih}x_t+b_{ih}+W_{hh}h_{t-1}+b_{hh}) htвҖӢ=tanh(WihвҖӢxtвҖӢ+bihвҖӢ+WhhвҖӢhtвҲ’1вҖӢ+bhhвҖӢ)

е…¶дёӯпјҡ

  • hth_thtвҖӢжҳҜж—¶й—ҙtж—¶еҲ»зҡ„йҡҗи—ҸзҠ¶жҖҒгҖӮ
  • xtx_txtвҖӢжҳҜж—¶й—ҙtж—¶еҲ»зҡ„иҫ“е…ҘгҖӮ
  • htвҲ’1h_{t-1}htвҲ’1вҖӢжҳҜж—¶й—ҙt-1ж—¶еҲ»зҡ„йҡҗи—ҸзҠ¶жҖҒгҖӮ
  • WihW_{ih}WihвҖӢе’ҢWhhW_{hh}WhhвҖӢеҲҶеҲ«жҳҜиҫ“е…Ҙxtx_txtвҖӢе’Ңйҡҗи—ҸзҠ¶жҖҒhtвҲ’1h_{t-1}htвҲ’1вҖӢзҡ„жқғйҮҚзҹ©йҳөгҖӮ
  • bihb_{ih}bihвҖӢе’Ңbhhb_{hh}bhhвҖӢеҲҶеҲ«жҳҜиҫ“е…Ҙxtx_txtвҖӢе’Ңйҡҗи—ҸзҠ¶жҖҒhtвҲ’1h_{t-1}htвҲ’1вҖӢзҡ„еҒҸзҪ®еҖјгҖӮ
  • tanh()tanh()tanh()жҳҜйқһзәҝжҖ§жҝҖжҙ»еҮҪж•°пјҢеҸҜд»Ҙз”ЁReLuReLuReLuд»ЈжӣҝгҖӮ

RNNзҡ„еҸӮж•°жңү:

  • input_sizeпјҡиҫ“е…Ҙxxxзҡ„зү№еҫҒж•°йҮҸгҖӮ
  • hidden_sizeпјҡйҡҗи—ҸеұӮhhhзҡ„зү№еҫҒж•°йҮҸгҖӮ
  • num_layersпјҡRNNеұӮзҡ„ж•°йҮҸпјҢnum_layers=2иЎЁзӨәз”ЁдёӨдёӘRNNеұӮеҪўжҲҗдёҖдёӘеӨҡеұӮRNNпјҢз¬¬дәҢдёӘRNNеұӮзҡ„иҫ“е…ҘжҳҜз¬¬дёҖдёӘRNNеұӮзҡ„иҫ“еҮәгҖӮй»ҳи®Өдёә1гҖӮ
  • nonlinearityпјҡйқһзәҝжҖ§еҮҪж•°пјҢеҸҜд»ҘжҳҜвҖңtanhвҖқжҲ–вҖңReLuвҖқгҖӮй»ҳи®ӨдёәвҖңtanhвҖқгҖӮ
  • biasпјҡжҳҜеҗҰдҪҝз”ЁеҒҸзҪ®еҖјгҖӮй»ҳи®ӨдёәTrueгҖӮ
  • batch_firstпјҡеҶіе®ҡиҫ“е…Ҙе’Ңиҫ“еҮәзҡ„ж јејҸгҖӮеҰӮжһңдёәTrueпјҢйӮЈд№ҲжҲ‘д»¬зҡ„иҫ“е…Ҙе’Ңиҫ“еҮәеј йҮҸзҡ„ж јејҸе°ұжҳҜ(batch,seq,feature)пјҢеҰӮжһңдёәFalseпјҢйӮЈд№ҲжҲ‘д»¬зҡ„иҫ“е…Ҙе’Ңиҫ“еҮәеј йҮҸзҡ„ж јејҸе°ұжҳҜ(seq,batch,feature)гҖӮй»ҳи®ӨдёәFalseгҖӮ
  • dropoutпјҡdropoutеҸӮж•°гҖӮй»ҳи®Өдёә0гҖӮ
  • bidirectionalпјҡжҳҜеҗҰдёәеҸҢеҗ‘RNNгҖӮеҰӮжһңдёәTrueеҲҷиЎЁзӨәдҪҝз”ЁеҸҢеҗ‘RNNгҖӮеҰӮжһңжҳҜеҸҢеҗ‘RNNпјҢйӮЈд№Ҳиҫ“еҮәе°ұжҳҜдёӨеҖҚзҡ„hidden_sizeгҖӮй»ҳи®ӨдёәFalseгҖӮ

RNNзҡ„иҫ“е…Ҙпјҡ

  • inputпјҡеҪ“batch_first=Falseж—¶иҫ“е…ҘеҪўзҠ¶дёә(LLL,NNN,HinH_{in}HinвҖӢ)пјҢеҪ“batch_first=Trueж—¶иҫ“е…ҘеҪўзҠ¶дёә(NNN,LLL,HinH_{in}HinвҖӢ)гҖӮ
  • h_0пјҡеҪўзҠ¶дёә(DвҲ—numlayersD*num_layersDвҲ—numlвҖӢayers,NNN,HoutH_{out}HoutвҖӢ)гҖӮ
    е…¶дёӯпјҡ
  • NNN = batch size
  • LLL = sequence length
  • DDD = еҚ•еҗ‘RNNж—¶дёә1пјҢеҸҢеҗ‘RNNж—¶дёә2
  • HinH_{in}HinвҖӢ = input_size
  • HoutH_{out}HoutвҖӢ = hidden_size

RNNзҡ„иҫ“еҮәпјҡ

  • outputпјҡеҪ“batch_first=Falseж—¶иҫ“е…ҘеҪўзҠ¶дёә(LLL,NNN,DвҲ—HoutD*H_{out}DвҲ—HoutвҖӢ)пјҢеҪ“batch_first=Trueж—¶иҫ“е…ҘеҪўзҠ¶дёә(LLL,NNN,DвҲ—HoutD*H_{out}DвҲ—HoutвҖӢ)гҖӮ
  • h_nпјҡеҪўзҠ¶дёә(DвҲ—numlayersD*num_layersDвҲ—numlвҖӢayers,NNN,HoutH_{out}HoutвҖӢ)гҖӮ

RNNзҡ„жқғйҮҚе’ҢеҒҸзҪ®зҹ©йҳө:

  • weight_ih_l[k]пјҡз¬¬kеұӮзҡ„иҫ“е…ҘжқғйҮҚзҹ©йҳөгҖӮеңЁK=0ж—¶еҪўзҠ¶дёә(hidden_size, input_size)пјҢеҗҰеҲҷеҪўзҠ¶дёә(hidden_size,num_directions*hidden_size)гҖӮ
  • weight_hh_l[k]пјҡз¬¬kеұӮзҡ„йҡҗи—ҸзҠ¶жҖҒжқғйҮҚзҹ©йҳөгҖӮеҪўзҠ¶дёә(hidden_size, hidden_size)гҖӮ
  • bias_ih_l[k]пјҡз¬¬kеұӮзҡ„иҫ“е…ҘеҒҸзҪ®еҖјзҹ©йҳөгҖӮеҪўзҠ¶дёә(hidden_size)гҖӮ
  • bias_hh_l[k]пјҡз¬¬kеұӮзҡ„йҡҗи—ҸзҠ¶жҖҒеҒҸзҪ®еҖјзҹ©йҳөгҖӮеҪўзҠ¶дёә(hidden_size)гҖӮ

1.2 еҸҢеҗ‘еҫӘзҺҜзҪ‘з»ң

1.2.1 еҸҢеҗ‘RNNз»“жһ„

еҚ•еҗ‘еҫӘзҺҜзҪ‘з»ңеҸӘиғҪдҫқжҚ®д№ӢеүҚж—¶еҲ»зҡ„ж—¶еәҸдҝЎжҒҜжқҘйў„жөӢдёӢдёҖж—¶еҲ»зҡ„иҫ“еҮәпјҢдҪҶеңЁжңүдәӣй—®йўҳдёӯпјҢеҪ“еүҚж—¶еҲ»зҡ„иҫ“еҮәдёҚд»…е’Ңд№ӢеүҚзҡ„зҠ¶жҖҒжңүе…іпјҢиҝҳеҸҜиғҪдёҺжңӘжқҘзҡ„зҠ¶жҖҒжңүе…ігҖӮ
жҜ”еҰӮиҰҒйў„жөӢдёҖеҸҘиҜқдёӯзјәеӨұзҡ„еҚ•иҜҚж—¶пјҢе°ұйңҖиҰҒеҗҢж—¶иҖғиҷ‘дёҠдёӢж–Үзҡ„еҶ…е®№гҖӮ
еҸҢеҗ‘RNNжңүдёӨдёӘRNNдёҠдёӢеҸ еҠ еңЁдёҖиө·з»„жҲҗпјҢиҫ“еҮәз”ұиҝҷдёӨдёӘRNNзҡ„зҠ¶жҖҒе…ұеҗҢеҶіе®ҡгҖӮ

еҸҢеҗ‘RNNзҡ„и®Ўз®—иҝҮзЁӢеҰӮдёӢпјҡ

  1. йҰ–е…ҲиҝӣиЎҢеүҚеҗ‘RNNпјҢиҫ“е…Ҙx1x_1x1вҖӢе’Ңh01h_0^1h01вҖӢпјҢиҫ“еҮәh11h_1^1h11вҖӢгҖӮ
  2. еҶҚж №жҚ®x2x_2x2вҖӢе’Ңh21h_2^1h21вҖӢи®Ўз®—еҫ—еҲ°h31h_3^1h31вҖӢпјҢд»ҘжӯӨзұ»жҺЁиҝӣиЎҢе®ҢеүҚеҗ‘RNNгҖӮ
  3. еҪ“еүҚеҗ‘RNNе®ҢжҲҗеҗҺи®Ўз®—еҸҚеҗ‘RNNпјҢеҸҚеҗ‘RNNжңүеҚ•зӢ¬зҡ„йҡҗи—ҸеұӮh02h_0^2h02вҖӢпјҢдё”еңЁеҸҚеҗ‘RNNдёӯиҫ“е…ҘinputжҳҜеҸҚеҗ‘иҫ“е…Ҙзҡ„гҖӮ
  4. йҰ–е…Ҳиҫ“е…Ҙx4x_4x4вҖӢе’Ңh02h_0^2h02вҖӢи®Ўз®—еҫ—еҲ°h12h_1^2h12вҖӢпјҢз„¶еҗҺз”ұh41h_4^1h41вҖӢе’Ңh12h_1^2h12вҖӢжӢјжҺҘеҫ—еҲ°жңҖз»Ҳиҫ“еҮәзҡ„з¬¬дёҖдёӘиҫ“еҮәy4y_4y4вҖӢ(еҸҜд»ҘзӣҙжҺҘжӢјжҺҘд№ҹеҸҜд»ҘзӣёеҠ еҫ—еҲ°y4y_4y4вҖӢ)гҖӮ
  5. еҶҚз”ұx3x_3x3вҖӢе’Ңh12h_1^2h12вҖӢеҫ—еҲ°h22h_2^2h22вҖӢпјҢз”ұh31h_3^1h31вҖӢе’Ңh22h_2^2h22вҖӢжӢјжҺҘеҫ—еҲ°иҫ“еҮәy3y_3y3вҖӢгҖӮ
  6. д»ҘжӯӨзұ»жҺЁиҝӣиЎҢе®ҢеҸҚеҗ‘RNNиҝҮзЁӢеҚіеҸҜгҖӮ

  • иҫ“е…Ҙпјҡ[x1,x2,x3,x4x_1,x_2,x_3,x_4x1вҖӢ,x2вҖӢ,x3вҖӢ,x4вҖӢ]

дҪҝз”Ёpytorchе®һзҺ°еҸҢеҗ‘RNNеҸӘйңҖиҰҒж”№еҸҳдј е…ҘRNNзҡ„еҸӮж•°bidirectional=TrueеҚіеҸҜгҖӮ

1.2.3 жүӢеҶҷеҸҢеҗ‘RNN

дәҢгҖҒй•ҝзҹӯжңҹи®°еҝҶзҪ‘з»ңпјҲLSTMпјү

дј з»ҹзҡ„еҫӘзҺҜзҘһз»ҸзҪ‘з»ңRNNиҷҪз„¶иғҪеӨҹиҒ”зі»дёҠдёӢж–Үзҡ„дҝЎжҒҜпјҢдҪҶжҳҜRNNзҡ„жўҜеәҰйңҖиҰҒйҖҡиҝҮж—¶й—ҙеҸҚеҗ‘дј ж’ӯ(Backpropagation Through Time)дј ж’ӯеҫҲй•ҝзҡ„ж—¶й—ҙжӯҘгҖӮеҪ“еәҸеҲ—й•ҝеәҰиҫғеӨ§ж—¶пјҢе°ұдјҡеҮәзҺ°жўҜеәҰж¶ҲеӨұжҲ–жўҜеәҰзҲҶзӮёзҡ„й—®йўҳгҖӮиҝҷз§Қй—®йўҳеҸҜиғҪдјҡеҜјиҮҙжҹҗдәӣдёҚеӨӘйҮҚиҰҒзҡ„дҝЎжҒҜеҜ№е…¶еҗҺз»ӯдҝЎжҒҜзҡ„йў„жөӢйҖ жҲҗеҪұе“ҚгҖӮ
жүҖд»ҘжҲ‘д»¬еј•е…ҘдәҶвҖңй•ҝзҹӯжңҹи®°еҝҶвҖқ(long-short-term memory, LSTM)е’ҢвҖңй—ЁжҺ§еҫӘзҺҜеҚ•е…ғвҖқ(gated recurrent unit,GRU)гҖӮ
дј з»ҹзҡ„RNNеҜ№дәҺиҫ“е…Ҙзҡ„йҮҚиҰҒжҖ§еҲӨж–ӯеҫҲеӣәе®ҡпјҢи¶Ҡж—©иҫ“е…Ҙзҡ„и¶ҠдёҚйҮҚиҰҒпјҢи¶Ҡжҷҡиҫ“е…Ҙзҡ„и¶ҠйҮҚиҰҒпјҢиҝҷжҳҫз„¶еӯҳеңЁдёҖе®ҡзҡ„й—®йўҳгҖӮиҖҢLSTMеҲҷжҳҜйҖҡиҝҮй—ЁжҺ§еҲ¶зҡ„ж–№ејҸе®һзҺ°дәҶеҜ№иҫ“е…ҘйҮҚиҰҒеәҰзҡ„жҺ§еҲ¶гҖӮ

2.1 LSTMз»“жһ„

еҺҹе§Ӣзҡ„RNNйҡҗи—ҸзҠ¶жҖҒеҸӘжңүдёҖдёӘhhhпјҢе®ғеҜ№дәҺзҹӯжңҹиҫ“е…Ҙйқһеёёж•Ҹж„ҹпјҢжүҖд»ҘжҲ‘д»¬зҡ„жғіжі•е°ұжҳҜеўһеҠ дёҖдёӘйҡҗи—ҸзҠ¶жҖҒcccз”ЁжқҘдҝқеӯҳй•ҝжңҹзҠ¶жҖҒпјҢжҲ‘д»¬жҠҠзҠ¶жҖҒcccз§°дёәеҖҷйҖүи®°еҝҶеҚ•е…ғпјҢиҝҷе°ұжҳҜLSTMгҖӮ
LSTMзҡ„з»“жһ„еҰӮдёӢеӣҫпјҡ

LSTMзҡ„е…¬ејҸеҰӮдёӢпјҡ

LSTMдёӯжңүдёүз§Қй—Ёпјҡiti_titвҖӢжҳҜиҫ“е…Ҙй—ЁпјҢе®ғжҺ§еҲ¶дҪҝз”ЁеӨҡе°‘жқҘиҮӘеҖҷйҖүи®°еҝҶеҚ•е…ғgtg_tgtвҖӢзҡ„ж•°жҚ®пјҢйҒ—еҝҳй—Ёftf_tftвҖӢжҺ§еҲ¶дҝқз•ҷеӨҡе°‘иҝҮеҺ»зҡ„и®°еҝҶе…ғctвҲ’1c_{t-1}ctвҲ’1вҖӢзҡ„еҶ…е®№пјҢиҫ“еҮәй—Ёoto_totвҖӢз”ЁдәҺеҶіе®ҡеҪ“еүҚж—¶еҲ»иҫ“еҮәе“ӘдәӣдҝЎжҒҜгҖӮ
е…¬ејҸдёӯзҡ„жҝҖжҙ»еҮҪж•°Пғ\sigmaПғ(еҚіsigmoid)е’Ңtanhtanhtanhзӣ®зҡ„йғҪжҳҜе°Ҷиҫ“еҮәйҷҗеҲ¶еңЁ[0,1]д№Ӣй—ҙгҖӮ

2.2 жүӢеҶҷLSTM

LSTMзҡ„е®һзҺ°дёҺRNNзұ»дјјпјҢдҪҶжҳҜйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢеӣ дёәжҲ‘д»¬еңЁit,ft,gt,oti_t,f_t,g_t,o_titвҖӢ,ftвҖӢ,gtвҖӢ,otвҖӢдёӯйғҪиҰҒе®һзҺ°WixtW_ix_tWiвҖӢxtвҖӢе’ҢWhxtW_hx_tWhвҖӢxtвҖӢпјҢжүҖд»ҘжҲ‘д»¬еҸҜд»Ҙе°ҶеӣӣдёӘжқғйҮҚзҹ©йҳөжӢјжҺҘеҲ°дёҖиө·пјҢз„¶еҗҺеҸӘиҝӣиЎҢдёҖж¬Ўзҹ©йҳөд№ҳжі•иҝҗз®—еҚіеҸҜгҖӮ

дёүгҖҒй—ЁжҺ§еҫӘзҺҜеҚ•е…ғпјҲGRUпјү

GRUзҡ„е®һзҺ°еҺҹзҗҶдёҺLSTMзұ»дјјпјҢеҸӘдёҚиҝҮGRUеҜ№LSTMеҒҡдәҶдёҖдәӣдјҳеҢ–пјҢеҮҸе°‘дәҶдёҖдәӣеҸӮж•°гҖӮ

3.1 GRUз»“жһ„
