жҖқи·ҜеҸҠз®—жі•пјҡ
жЁЎжӢҹзҹ©йҳөзҡ„з”ҹжҲҗгҖӮжҢүз…§иҰҒжұӮпјҢеҲқе§ӢдҪҚзҪ®и®ҫдёәзҹ©йҳөзҡ„е·ҰдёҠи§’пјҢеҲқе§Ӣж–№еҗ‘и®ҫдёәеҗ‘еҸігҖӮиӢҘдёӢдёҖжӯҘзҡ„дҪҚзҪ®и¶…еҮәзҹ©йҳөиҫ№з•ҢпјҢжҲ–иҖ…жҳҜд№ӢеүҚи®ҝй—®иҝҮзҡ„дҪҚзҪ®пјҢеҲҷйЎәж—¶й’Ҳж—ӢиҪ¬пјҢиҝӣе…ҘдёӢдёҖдёӘж–№еҗ‘гҖӮеҰӮжӯӨеҸҚеӨҚзӣҙиҮіеЎ«е…Ҙn^2дёӘе…ғзҙ гҖӮ
и®° matrix дёәз”ҹжҲҗзҡ„зҹ©йҳөпјҢе…¶еҲқе§Ӣе…ғзҙ и®ҫдёә 0гҖӮз”ұдәҺеЎ«е…Ҙзҡ„е…ғзҙ еқҮдёәжӯЈж•°пјҢжҲ‘д»¬еҸҜд»ҘеҲӨж–ӯеҪ“еүҚдҪҚзҪ®зҡ„е…ғзҙ еҖјпјҢиӢҘдёҚдёә 0пјҢеҲҷиҜҙжҳҺе·Із»Ҹи®ҝй—®иҝҮжӯӨдҪҚзҪ®гҖӮ

еӨҚжқӮеәҰеҲҶжһҗпјҡ

  • ж—¶й—ҙеӨҚжқӮеәҰпјҡO(n^2)пјҢе…¶дёӯ n жҳҜз»ҷе®ҡзҡ„жӯЈж•ҙж•°гҖӮзҹ©йҳөзҡ„еӨ§е°ҸжҳҜ nГ—nпјҢйңҖиҰҒеЎ«е…Ҙзҹ©йҳөдёӯзҡ„жҜҸдёӘе…ғзҙ гҖӮ
  • з©әй—ҙеӨҚжқӮеәҰпјҡO(1)гҖӮйҷӨдәҶиҝ”еӣһзҡ„зҹ©йҳөд»ҘеӨ–пјҢз©әй—ҙеӨҚжқӮеәҰжҳҜеёёж•°гҖӮ

ж–№жі•дәҢпјҡжҢүеұӮжЁЎжӢҹ

жҖқи·ҜдёҺз®—жі•пјҡ
еҸҜд»Ҙе°Ҷзҹ©йҳөзңӢжҲҗиӢҘе№ІеұӮпјҢйҰ–е…ҲеЎ«е…Ҙзҹ©йҳөжңҖеӨ–еұӮзҡ„е…ғзҙ пјҢе…¶ж¬ЎеЎ«е…Ҙзҹ©йҳөж¬ЎеӨ–еұӮзҡ„е…ғзҙ пјҢзӣҙеҲ°еЎ«е…Ҙзҹ©йҳөжңҖеҶ…еұӮзҡ„е…ғзҙ гҖӮ
е®ҡд№үзҹ©йҳөзҡ„з¬¬ k еұӮжҳҜеҲ°жңҖиҝ‘иҫ№з•Ңи·қзҰ»дёә k зҡ„жүҖжңүйЎ¶зӮ№гҖӮдҫӢеҰӮпјҢдёӢеӣҫзҹ©йҳөжңҖеӨ–еұӮе…ғзҙ йғҪжҳҜз¬¬ 1 еұӮпјҢж¬ЎеӨ–еұӮе…ғзҙ йғҪжҳҜз¬¬ 2 еұӮпјҢжңҖеҶ…еұӮе…ғзҙ йғҪжҳҜз¬¬ 3 еұӮгҖӮ

еҜ№дәҺжҜҸеұӮпјҢд»Һе·ҰдёҠж–№ејҖе§Ӣд»ҘйЎәж—¶й’Ҳзҡ„йЎәеәҸеЎ«е…ҘжүҖжңүе…ғзҙ гҖӮеҒҮи®ҫеҪ“еүҚеұӮзҡ„е·ҰдёҠи§’дҪҚдәҺ (top,left)пјҢеҸідёӢи§’дҪҚдәҺ (bottom,right)пјҢжҢүз…§еҰӮдёӢйЎәеәҸеЎ«е…ҘеҪ“еүҚеұӮзҡ„е…ғзҙ гҖӮ

д»Һе·ҰеҲ°еҸіеЎ«е…ҘдёҠдҫ§е…ғзҙ пјҢдҫқж¬Ўдёә (top,left) еҲ° (top,right)гҖӮ

д»ҺдёҠеҲ°дёӢеЎ«е…ҘеҸідҫ§е…ғзҙ пјҢдҫқж¬Ўдёә (top+1,right) еҲ° (bottom,right)гҖӮ

еҰӮжһң left<right дё” top<bottomпјҢеҲҷд»ҺеҸіеҲ°е·ҰеЎ«е…ҘдёӢдҫ§е…ғзҙ пјҢдҫқж¬Ўдёә (bottom,rightвҲ’1) еҲ° (bottom,left+1)пјҢд»ҘеҸҠд»ҺдёӢеҲ°дёҠеЎ«е…Ҙе·Ұдҫ§е…ғзҙ пјҢдҫқж¬Ўдёә (bottom,left) еҲ° (top+1,left)гҖӮ

еЎ«е®ҢеҪ“еүҚеұӮзҡ„е…ғзҙ д№ӢеҗҺпјҢе°Ҷ left е’Ң top еҲҶеҲ«еўһеҠ  1пјҢе°Ҷ right е’Ң bottom еҲҶеҲ«еҮҸе°‘ 1пјҢиҝӣе…ҘдёӢдёҖеұӮз»§з»ӯеЎ«е…Ҙе…ғзҙ пјҢзӣҙеҲ°еЎ«е®ҢжүҖжңүе…ғзҙ дёәжӯўгҖӮ

  • з©әй—ҙеӨҚжқӮеәҰпјҡO(1)гҖӮйҷӨдәҶиҝ”еӣһзҡ„зҹ©йҳөд»ҘеӨ–пјҢз©әй—ҙеӨҚжқӮеәҰжҳҜеёёж•°гҖӮ
