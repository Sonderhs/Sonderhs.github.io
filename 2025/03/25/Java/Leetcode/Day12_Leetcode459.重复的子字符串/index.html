жңҖй•ҝзӣёзӯүеүҚеҗҺзјҖзҡ„й•ҝеәҰдёәпјҡnext[len - 1] + 1гҖӮ(иҝҷйҮҢзҡ„nextж•°з»„жҳҜд»Ҙз»ҹдёҖеҮҸдёҖзҡ„ж–№ејҸи®Ўз®—зҡ„пјҢеӣ жӯӨйңҖиҰҒ+1пјҢдёӨз§Қи®Ўз®—nextж•°з»„зҡ„е…·дҪ“еҢәеҲ«зңӢиҝҷйҮҢпјҡеӯ—з¬ҰдёІпјҡKMPз®—жі•зІҫи®І (opens new window))

ж•°з»„й•ҝеәҰдёәпјҡlenгҖӮ

len - (next[len - 1] + 1) жҳҜжңҖй•ҝзӣёзӯүеүҚеҗҺзјҖдёҚеҢ…еҗ«зҡ„еӯҗдёІзҡ„й•ҝеәҰгҖӮ

еҰӮжһңlen % (len - (next[len - 1] + 1)) == 0 пјҢеҲҷиҜҙжҳҺж•°з»„зҡ„й•ҝеәҰжӯЈеҘҪеҸҜд»Ҙиў« жңҖй•ҝзӣёзӯүеүҚеҗҺзјҖдёҚеҢ…еҗ«зҡ„еӯҗдёІзҡ„й•ҝеәҰ ж•ҙйҷӨ пјҢиҜҙжҳҺиҜҘеӯ—з¬ҰдёІжңүйҮҚеӨҚзҡ„еӯҗеӯ—з¬ҰдёІгҖӮ

жү“еҚ°ж•°з»„

next[len - 1] = 7пјҢnext[len - 1] + 1 = 8пјҢ8е°ұжҳҜжӯӨж—¶еӯ—з¬ҰдёІasdfasdfasdfзҡ„жңҖй•ҝзӣёеҗҢеүҚеҗҺзјҖзҡ„й•ҝеәҰгҖӮ
