е®һйҷ…иҝҗиЎҢж—¶е·®дёҚеӨҡпјҡ

дёҖгҖҒBitFit

1.1 BitFitеҺҹзҗҶ

BitFitжҖқжғіпјҡеңЁиҝӣиЎҢи®ӯз»ғж—¶еҸӘи®ӯз»ғдёҖйғЁеҲҶеҸӮж•°пјҢBitFitйҖүжӢ©е°ұжҳҜеҸӘи®ӯз»ғжЁЎеһӢеҸӮж•°йҮҢйқўзҡ„жүҖжңүbiasйғЁеҲҶпјҢд№ҹе°ұжҳҜеҒҸзҪ®еҖјпјҢиҖҢеҶ»з»“жүҖжңүе…¶д»–еҸӮж•°пјҢд»ҺиҖҢеҮҸе°‘и®Ўз®—е’ҢеӯҳеӮЁжҲҗжң¬пјҢеҗҢж—¶дҝқжҢҒиҫғеҘҪзҡ„дёӢжёёд»»еҠЎжҖ§иғҪ
е®һзҺ°ж–№жі•пјҡеҸӘйңҖиҰҒжҠҠжүҖжңүйқһbiasйғЁеҲҶзҡ„жҳҜеҗҰйңҖиҰҒи®Ўз®—жўҜеәҰи®ҫзҪ®дёәfaseеҚіеҸҜ

1.2 д»Јз Ғе®һзҺ°

з»ҸиҝҮдјҳеҢ–еҗҺзҡ„жЁЎеһӢеҸӮж•°йҮҸзәҰдёә544768дёӘпјҢзӣёеҪ“дәҺдјҳеҢ–еүҚзҡ„еҸӮж•°йҮҸ1303111680зҡ„0.418%
жҳҫеӯҳеҚ з”ЁзҺҮд№ҹжҳҺжҳҫдёӢйҷҚ

1.3 BitFitзҡ„дјҳзјәзӮ№

дјҳзӮ№пјҡ

  • й«ҳж•ҲжҖ§пјҡзӣёжҜ”е…ЁеҸӮж•°еҫ®и°ғпјҢи®Ўз®—йҮҸе’ҢеӯҳеӮЁйңҖжұӮжӣҙдҪҺпјҢйҖӮеҗҲиө„жәҗеҸ—йҷҗзҺҜеўғпјҲеҰӮз§»еҠЁз«ҜгҖҒеөҢе…ҘејҸи®ҫеӨҮпјүгҖӮ
  • йҖӮй…ҚжҖ§ејәпјҡиғҪеӨҹйҖӮз”ЁдәҺеҗ„з§ҚдёӢжёёд»»еҠЎпјҢиҖҢж— йңҖеӯҳеӮЁеӨҡдёӘе®Ңж•ҙзҡ„еҫ®и°ғжЁЎеһӢгҖӮ
  • иҝҒз§»жҖ§ејәпјҡз”ұдәҺеҹәзЎҖжЁЎеһӢзҡ„жқғйҮҚжңӘж”№еҸҳпјҢеҸҜд»Ҙеҝ«йҖҹеңЁеӨҡдёӘд»»еҠЎд№Ӣй—ҙеҲҮжҚўгҖӮ

зјәзӮ№пјҡ

  • жҖ§иғҪеҸҜиғҪз•ҘйҖҠдәҺе…ЁеҸӮж•°еҫ®и°ғпјҡиҷҪз„¶еңЁи®ёеӨҡд»»еҠЎдёҠиЎЁзҺ°иүҜеҘҪпјҢдҪҶеңЁжҹҗдәӣеӨҚжқӮд»»еҠЎдёҠпјҢBitFit еҸҜиғҪжҜ”е…ЁеҸӮж•°еҫ®и°ғз•Ҙе·®гҖӮ
  • йҖӮз”ЁиҢғеӣҙжңүйҷҗпјҡйҖӮз”ЁдәҺ NLP д»»еҠЎпјҢдҪҶеңЁи®Ўз®—жңәи§Ҷи§үзӯүйўҶеҹҹзҡ„ж•Ҳжһңд»ҚйңҖжӣҙеӨҡз ”з©¶гҖӮ

дәҢгҖҒPrompt-Tuning

2.1 Prompt-TuningеҺҹзҗҶ

Prompt-TuningжҖқжғіпјҡеҶ»з»“дё»жЁЎеһӢе…ЁйғЁеҸӮж•°пјҢеңЁи®ӯз»ғж•°жҚ®еүҚеҠ е…ҘдёҖе°Ҹж®өPromptпјҢеҸӘи®ӯз»ғPromptзҡ„иЎЁзӨәеұӮпјҢеҚідёҖдёӘEmbeddingжЁЎеқ—гҖӮе…¶дёӯпјҢPromptеҸҲеӯҳеңЁдёӨз§ҚеҪўејҸпјҢдёҖз§ҚжҳҜhard promptпјҢдёҖз§ҚжҳҜsoft promptпјҢpromptзӣёеҪ“дәҺжҳҜдёҖз§ҚжҸҗзӨә

  • hard promptпјҡpromptеҶ…е®№дҪҝжҲ‘д»¬дәәдёәе®ҡд№үзҡ„
  • soft promptпјҡдёҚжҢҮе®ҡpromptпјҢи®©жЁЎеһӢиҮӘжҲ‘еӯҰд№ еҫ—еҲ°

е®һзҺ°ж–№жі•пјҡеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёpeftеә“иҝӣиЎҢе®һзҺ°

2.3 Prompt-Tuningзҡ„дјҳзјәзӮ№

дјҳзӮ№пјҡ

  • жһҒе°‘зҡ„и®ӯз»ғеҸӮж•°пјҡд»…жӣҙж–°е°‘йҮҸ Prompt еөҢе…ҘпјҢйҖӮз”ЁдәҺи¶…еӨ§жЁЎеһӢгҖӮ
  • дҪҺи®Ўз®—жҲҗжң¬пјҡжҜ”е…ЁеҸӮж•°еҫ®и°ғе’Ң Adapter ж–№ејҸжӣҙиҠӮзңҒиө„жәҗгҖӮ
  • д»»еҠЎйҖӮй…ҚзҒөжҙ»пјҡдёҚеҗҢд»»еҠЎеҸҜдҪҝз”ЁдёҚеҗҢзҡ„ PromptпјҢж— йңҖеӯҳеӮЁеӨҡдёӘеӨ§жЁЎеһӢгҖӮ

зјәзӮ№пјҡ

  • жҜ”е…ЁеҸӮж•°еҫ®и°ғз•ҘйҖҠиүІпјҡPrompt еҸӘиғҪеҪұе“Қиҫ“е…ҘпјҢеҸҜиғҪдёҚеҰӮ Adapter и°ғж•ҙ Transformer з»“жһ„жқҘеҫ—ејәеӨ§гҖӮ
  • еҜ№е°ҸжЁЎеһӢж•ҲжһңжңүйҷҗпјҡеңЁе°ҸеһӢ TransformerпјҲеҰӮ BERT-baseпјүдёҠеҸҜиғҪдёҚеҰӮе…ЁеҸӮж•°еҫ®и°ғж•ҲжһңеҘҪгҖӮ

дёүгҖҒP-Tuning

3.1 P-TuningеҺҹзҗҶ

P-TuningжҖқжғіпјҡеңЁPrompt-Tuningзҡ„еҹәзЎҖдёҠпјҢеҜ№PromptйғЁеҲҶиҝӣиЎҢдёҖжӯҘзҡ„зј–з Ғи®Ўз®—пјҢеҠ йҖҹж”¶ж•ӣгҖӮе…·дҪ“жқҘиҜҙпјҢPEFTдёӯж”ҜжҢҒдёӨз§Қзј–з Ғж–№ејҸпјҢдёҖз§ҚжҳҜLSTMпјҢдёҖз§ҚжҳҜMLPгҖӮдёҺPrompt-TuningдёҚеҗҢзҡ„жҳҜпјҢPromptзҡ„еҪўејҸеҸӘжңүSoft Prompt

е®һзҺ°ж–№жі•пјҡеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёpeftеә“иҝӣиЎҢе®һзҺ°

3.2 д»Јз Ғе®һзҺ°

з»ҸиҝҮдјҳеҢ–еҗҺеҸҜи®ӯз»ғеҸӮж•°жңү5,267,456дёӘпјҢзӣёеҪ“дәҺеҺҹжқҘеҸӮж•°1,308,379,136дёӘзҡ„0.4026%

3.3 P-Tuningзҡ„дјҳзјәзӮ№

дјҳзӮ№пјҡ

  • еҸӮж•°й«ҳж•ҲпјҲParameter-EfficientпјүпјҡеҸӘдјҳеҢ– Soft Prompt + MLP/LSTMпјҢеҶ»з»“ йў„и®ӯз»ғжЁЎеһӢзҡ„е…ЁйғЁеҸӮж•°пјҢи®ӯз»ғжҲҗжң¬иҝңдҪҺдәҺе…ЁеҸӮж•°еҫ®и°ғпјҲFull Fine-TuningпјүгҖӮ
  • жҜ” Prompt Tuning жӣҙејәеӨ§пјҡзӣёжҜ”дәҺ Prompt TuningпјҲд»…дјҳеҢ– Soft Prompt еөҢе…ҘпјүпјҢP-Tuning йҮҮз”Ё MLP/LSTM з”ҹжҲҗ PromptпјҢе…·жңүжӣҙејәзҡ„иЎЁзӨәиғҪеҠӣгҖӮеңЁе°Ҹж ·жң¬еӯҰд№ пјҲFew-Shot Learningпјүд»»еҠЎдёҠиЎЁзҺ°жӣҙдҪігҖӮ
  • йҖӮз”ЁдәҺ Few-Shot / Zero-Shot еӯҰд№ пјҡйҖӮз”ЁдәҺ дҪҺиө„жәҗд»»еҠЎпјҢеңЁе°‘йҮҸж•°жҚ®дёҠиғҪиҺ·еҫ—жҜ”е…ЁеҸӮж•°еҫ®и°ғжӣҙдјҳзҡ„ж•ҲжһңгҖӮеҜ№ Prompt и®ҫи®Ўж•Ҹж„ҹзҡ„д»»еҠЎпјҲеҰӮж–Үжң¬еҲҶзұ»гҖҒй—®зӯ”зӯүпјүзү№еҲ«жңүж•ҲгҖӮ
  • зҒөжҙ»жҖ§ејәпјҡз”ұдәҺдҪҝз”Ё MLP/LSTM з”ҹжҲҗ PromptпјҢP-Tuning еҸҜд»ҘеӯҰд№ жӣҙеӨҚжқӮзҡ„ Prompt иЎЁиҫҫпјҢиҖҢдёҚеғҸжүӢе·Ҙ Prompt йӮЈж ·еҸ—йҷҗдәҺиҮӘз„¶иҜӯиЁҖз»“жһ„гҖӮ
  • и®Ўз®—ж•ҲзҺҮй«ҳпјҡеҸӘдјҳеҢ– Soft Prompt + MLP/LSTMпјҢеҸӮж•°йҮҸиҝңе°ҸдәҺе…ЁеҸӮж•°еҫ®и°ғпјҢи®Ўз®—йңҖжұӮдҪҺпјҢйҖӮеҗҲ иө„жәҗеҸ—йҷҗзҺҜеўғпјҲеҰӮдә‘з«Ҝ API и°ғз”ЁпјүгҖӮ

зјәзӮ№пјҡ

  • жҜ”е…ЁеҸӮж•°еҫ®и°ғпјҲFine-Tuningпјүз•ҘйҖҠиүІпјҡеңЁеӨ§и§„жЁЎж•°жҚ®йӣҶдёҠпјҢе…ЁеҸӮж•°еҫ®и°ғд»Қз„¶жҳҜжңҖдјҳж–№жЎҲпјҢP-Tuning еҸҜиғҪеңЁжҹҗдәӣд»»еҠЎдёҠж— жі•и¶…и¶Ҡ Full Fine-Tuning зҡ„жҖ§иғҪгҖӮдё»иҰҒйҖӮз”ЁдәҺ дҪҺиө„жәҗд»»еҠЎ жҲ– и®Ўз®—еҸ—йҷҗеңәжҷҜгҖӮ
  • дҫқиө–жЁЎеһӢжһ¶жһ„пјҡSoft Prompt еҸӘиғҪеҪұе“Қиҫ“е…ҘйғЁеҲҶпјҢиҖҢ LoRA / Adapter зӯүж–№жі• еҸҜд»Ҙи°ғж•ҙ Transformer еҶ…йғЁеҸӮж•°пјҢеӣ жӯӨеңЁжҹҗдәӣд»»еҠЎдёҠпјҢP-Tuning еҸҜиғҪдёҚеҰӮ LoRA зӯүж–№жі•ејәеӨ§гҖӮйҖӮз”ЁдәҺ NLP д»»еҠЎпјҢдҪҶеңЁ и®Ўз®—жңәи§Ҷи§үпјҲCVпјү д»»еҠЎдёҠдёҚеҰӮ LoRA / Adapter еёёи§ҒгҖӮ

4.1 Prefix-TuningеҺҹзҗҶ

  • еӣ жӯӨпјҢеҸҜд»ҘйҖҡиҝҮзҹ©йҳөеҲҶи§Јзҡ„еҪўејҸпјҢе°ҶеҺҹжң¬иҰҒжӣҙж–°зҡ„еӨ§зҹ©йҳөеҸҳжҲҗдёӨдёӘе°Ҹзҹ©йҳө
  • жқғйҮҚжӣҙж–°пјҡW = W + О”W = W + BA

  • и®ӯз»ғе®ҢжҲҗеҗҺпјҢеҸҜд»Ҙе°ҶдёӨдёӘдҪҺз§©зҹ©йҳөдёҺеҺҹе§ӢжЁЎеһӢдёӯзҡ„жқғйҮҚиҝӣиЎҢеҗҲе№¶пјҢеҗҲе№¶еҗҺзҡ„жЁЎеһӢдёҺеҺҹе§ӢжЁЎеһӢж— ејӮпјҢйҒҝе…ҚдәҶжҺЁзҗҶжңҹй—ҙPromptзі»еҲ—ж–№жі•еёҰжқҘзҡ„йўқеӨ–и®Ўз®—йҮҸ

  • еҸҜиғҪдёҚйҖӮз”ЁдәҺжүҖжңүд»»еҠЎ
