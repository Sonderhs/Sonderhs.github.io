1.1 Redisз®Җд»Ӣ

  • RedisжҳҜдёҖдёӘеҹәдәҺеҶ…еӯҳзҡ„key-valueз»“жһ„ж•°жҚ®еә“
  • Redisзү№зӮ№пјҡ
    • еҹәдәҺеҶ…еӯҳеӯҳеӮЁпјҢиҜ»еҶҷжҖ§иғҪй«ҳ
    • йҖӮеҗҲеӯҳеӮЁзғӯзӮ№ж•°жҚ®пјҲзғӯзӮ№е•Ҷе“ҒгҖҒиө„и®ҜгҖҒж–°й—»пјү
    • дјҒдёҡеә”з”Ёе№ҝжіӣ
  • е®ҳзҪ‘пјҡhttps://redis.io/
  • дёӯж–ҮзҪ‘пјҡhttps://www.redis.net.cn/

1.2 RedisдёӢиҪҪдёҺе®үиЈ…

  • Redisе®үиЈ…еҢ…еҲҶдёәWindowsе’ҢLinuxзүҲжң¬
    • WindowsзүҲжң¬пјҡhttps://github.com/microsoftarchive/redis/releases
    • LinuxзүҲжң¬пјҡhttps://download.redis.io/releases/
  • еӣ дёәRedisжҳҜз»ҝиүІиҪҜд»¶пјҢжүҖд»ҘдёҚйңҖиҰҒе®үиЈ…пјҢзӣҙжҺҘи§ЈеҺӢеҚіеҸҜгҖӮ

1.3 Redisй…ҚзҪ®дёҺеҗҜеҠЁ

  • Redisй…ҚзҪ®ж–Үд»¶пјҡredis.windows.conf
    • еҸҜд»Ҙдҝ®ж”№й…ҚзҪ®ж–Үд»¶дёӯзҡ„йҖүйЎ№жқҘи°ғж•ҙRedisзҡ„иЎҢдёә
    • еёёз”Ёй…ҚзҪ®йҖүйЎ№еҢ…жӢ¬пјҡ
      • bindпјҡз»‘е®ҡIPең°еқҖпјҢй»ҳи®Өз»‘е®ҡеҲ°жң¬ең°IP
      • portпјҡRedisжңҚеҠЎз«ҜеҸЈпјҢй»ҳи®Ө6379
      • requirepassпјҡи®ҫзҪ®и®ҝй—®еҜҶз Ғ
  • иҝҗиЎҢпјҡ
    • еңЁе®үиЈ…зӣ®еҪ•дёӢеҗҜеҠЁcmdпјҢе№¶иҫ“е…Ҙredis-server.exe redis.windows.confеҚіеҸҜ
    • ctrl+CеҸҜд»ҘеҒңжӯўRedisжңҚеҠЎ
  • Redisй»ҳи®ӨжІЎжңүеҜҶз ҒпјҢеҸҜд»ҘеңЁredis.windows.confдёӯи®ҫзҪ®еҜҶз Ғ
    • дҝ®ж”№й…ҚзҪ®ж–Үд»¶дёӯзҡ„requirepassйҖүйЎ№пјҢеҸ–ж¶ҲжіЁйҮҠе№¶и®ҫзҪ®еҜҶз Ғ
    • йҮҚеҗҜRedisжңҚеҠЎеҗҺпјҢдҪҝз”Ёredis-cliиҝһжҺҘж—¶йңҖиҰҒиҫ“е…ҘеҜҶз ҒпјҢж јејҸдёә-a еҜҶз Ғ
  • иҝһжҺҘеҲ°Redisпјҡ
    • ж–№ејҸдёҖпјҡеңЁе®үиЈ…зӣ®еҪ•дёӢеҗҜеҠЁcmdпјҢе№¶иҫ“е…Ҙredis-cli.exeеҚіеҸҜ
    • ж–№ејҸдәҢпјҡдҪҝз”ЁеӣҫеҪўеҢ–з•ҢйқўAnother Redis Desktop Managerй“ҫжҺҘ

дәҢгҖҒRedisж•°жҚ®зұ»еһӢ

2.1 Redisж•°жҚ®зұ»еһӢжҰӮиҝ°

  • Redisж”ҜеӯҳеӮЁзҡ„жҳҜkey-valueз»“жһ„зҡ„ж•°жҚ®пјҢе…¶дёӯkeyжҳҜеӯ—з¬ҰдёІзұ»еһӢпјҢvalueеҸҜд»ҘжҳҜеӨҡз§Қж•°жҚ®зұ»еһӢпјҡ
    • еӯ—з¬ҰдёІпјҲstringпјү
    • е“ҲеёҢпјҲhashпјү
    • еҲ—иЎЁпјҲlistпјү
    • йӣҶеҗҲпјҲsetпјү
    • жңүеәҸйӣҶеҗҲпјҲsorted set/zsetпјү

2.2 Redisж•°жҚ®зұ»еһӢзү№зӮ№

  • еӯ—з¬ҰдёІ(string)пјҡжҷ®йҖҡеӯ—з¬ҰдёІпјҢRedisдёӯжңҖз®ҖеҚ•зҡ„ж•°жҚ®зұ»еһӢ
  • е“ҲеёҢ(hash)пјҡд№ҹеҸ«ж•ЈеҲ—пјҢзұ»дјјдәҺJavaдёӯзҡ„HashMapз»“жһ„пјҢеҸҜд»ҘзҗҶи§Јдёә(key-(key-vaue))еҜ№зҡ„йӣҶеҗҲ
  • еҲ—иЎЁ(list)пјҡжҢүз…§жҸ’е…ҘйЎәеәҸжҺ’еәҸпјҢеҸҜд»ҘжңүйҮҚеӨҚе…ғзҙ пјҢзұ»дјјдәҺJavaдёӯзҡ„LinkedList
  • йӣҶеҗҲ(set)пјҡж— еәҸдё”е”ҜдёҖзҡ„е…ғзҙ йӣҶеҗҲпјҢзұ»дјјдәҺJavaдёӯзҡ„HashSet
  • жңүеәҸйӣҶеҗҲ(sorted set/zset)пјҡжңүеәҸдё”е”ҜдёҖзҡ„е…ғзҙ йӣҶеҗҲпјҢжҜҸдёӘе…ғзҙ йғҪжңүдёҖдёӘеҲҶж•°(score)жқҘеҶіе®ҡйЎәеәҸпјҢзұ»дјјдәҺJavaдёӯзҡ„TreeSet
