HashMap

еә•еұӮеҺҹзҗҶи§ҒпјҡJavaйӣҶеҗҲиҝӣйҳ¶

1. HashMapз®Җд»Ӣ

  • HashMapеӯҳеӮЁж•°жҚ®йҮҮз”ЁдәҶе“ҲеёҢиЎЁзҡ„з»“жһ„пјҢеә•еұӮдҪҝз”ЁдёҖз»ҙж•°з»„+еҚ•еҗ‘й“ҫиЎЁ+зәўй»‘ж ‘иҝӣиЎҢkey-valueеҜ№зҡ„еӯҳеӮЁгҖӮ
  • зәўй»‘ж ‘еҮәзҺ°зҡ„ж—¶жңәпјҡеҪ“жҹҗдёӘзҙўеј•дҪҚзҪ®iдёҠзҡ„й“ҫиЎЁй•ҝеәҰиҫҫеҲ°8пјҢдё”ж•°з»„й•ҝеәҰи¶…иҝҮ64ж—¶пјҢжӯӨзҙўеј•дҪҚзҪ®дёҠзҡ„е…ғзҙ иҰҒд»ҺеҚ•еҗ‘й“ҫиЎЁж”№дёәзәўй»‘ж ‘гҖӮеҺҹеӣ пјҡзәўй»‘ж ‘иҝӣиЎҢput()/get()/remove()ж“ҚдҪңзҡ„ж—¶й—ҙеӨҚжқӮеәҰдёәO(logn)пјҢжҜ”еҚ•еҗ‘й“ҫиЎЁзҡ„ж—¶й—ҙеӨҚжқӮеәҰO(n)еҘҪгҖӮ
  • зәўй»‘ж ‘йҖҖеҢ–жҲҗеҚ•еҗ‘й“ҫиЎЁпјҡеҰӮжһңзҙўеј•iдҪҚзҪ®жҳҜзәўй»‘ж ‘пјҢеҪ“дёҚж–ӯеҲ йҷӨе…ғзҙ зҡ„жғ…еҶөдёӢпјҢiеӨ„е…ғзҙ дёӘж•°дҪҺдәҺ6ж—¶пјҢзәўй»‘ж ‘дјҡйҖҖеҢ–жҲҗеҚ•еҗ‘й“ҫиЎЁгҖӮ

2. HashMapзҡ„жү©е®№жңәеҲ¶

  • и§ҰеҸ‘жқЎд»¶пјҡ еҪ“HashMapдёӯеӯҳеӮЁзҡ„й”®еҖјеҜ№ж•°йҮҸи¶…иҝҮдёҖдёӘйҳҲеҖјж—¶пјҢе°ұдјҡи§ҰеҸ‘жү©е®№гҖӮиҝҷдёӘйҳҲеҖјйҖҡеёёжҳҜеҪ“еүҚе®№йҮҸ * еҠ иҪҪеӣ еӯҗпјҲloadFactorпјҢй»ҳи®Ө0.75пјү
  • жү©е®№иҝҮзЁӢпјҡ
    • еҲӣе»әж–°ж•°з»„пјҡ еҲӣе»әдёҖдёӘж–°зҡ„гҖҒжӣҙеӨ§зҡ„ж•°з»„пјҢйҖҡеёёжҳҜеҪ“еүҚж•°з»„е®№йҮҸзҡ„дёӨеҖҚгҖӮ
    • йҮҚж–°и®Ўз®—е“ҲеёҢдёҺиҝҒз§»пјҲRehashingпјүпјҡ йҒҚеҺҶж—§ж•°з»„дёӯзҡ„жҜҸдёҖдёӘй”®еҖјеҜ№пјҢйҮҚж–°и®Ўз®—е®ғд»¬еңЁж–°ж•°з»„дёӯзҡ„дҪҚзҪ®пјҲеӣ дёәж•°з»„й•ҝеәҰеҸҳдәҶпјҢе“ҲеёҢжҳ е°„зҡ„зҙўеј•д№ҹдјҡеҸҳпјүпјҢз„¶еҗҺе°Ҷе®ғд»¬ж”ҫе…Ҙж–°ж•°з»„зҡ„еҜ№еә”дҪҚзҪ®гҖӮиҝҷдёӘиҝҮзЁӢеҫҲйҮҚиҰҒпјҢеӣ дёәеҗҢдёҖдёӘе…ғзҙ еңЁж–°ж—§ж•°з»„дёӯзҡ„жЎ¶пјҲbucketпјүзҙўеј•еҫҲеҸҜиғҪдёҚеҗҢгҖӮ
  • зӣ®зҡ„пјҡ
    • еҮҸе°‘е“ҲеёҢеҶІзӘҒпјҡ йҖҡиҝҮжү©еӨ§ж•°з»„е®№йҮҸпјҢдҪҝеҫ—й”®еҖјеҜ№иғҪжӣҙеқҮеҢҖең°еҲҶеёғпјҢд»ҺиҖҢеҮҸе°‘е“ҲеёҢеҶІзӘҒпјҢдҝқжҢҒHashMapзҡ„жҹҘжүҫгҖҒжҸ’е…Ҙзӯүж“ҚдҪңзҡ„ж•ҲзҺҮпјҲжҺҘиҝ‘O(1)пјүгҖӮ
    • дҝқжҢҒжҖ§иғҪпјҡ еҰӮжһңдёҚжү©е®№пјҢеҪ“е…ғзҙ и¶ҠжқҘи¶ҠеӨҡж—¶пјҢй“ҫиЎЁпјҲжҲ–зәўй»‘ж ‘пјүдјҡи¶ҠжқҘи¶Ҡй•ҝпјҢжҖ§иғҪдјҡдёӢйҷҚгҖӮ

3. TreeMapе’ҢHashMapзҡ„еҢәеҲ«

  • е®һзҺ°еҺҹзҗҶпјҡ
    • HashMapеҹәдәҺhashиЎЁе®һзҺ°пјҢи®Ўз®—й”®зҡ„hashеҖјжқҘзЎ®е®ҡеӯҳеӮЁдҪҚзҪ®гҖӮй»ҳи®Өжғ…еҶөдёӢпјҢhashmapдёҚдҝқиҜҒе…ғзҙ зҡ„еӯҳеӮЁйЎәеәҸпјҢд»Һjava8д№ӢеҗҺпјҢhй“ҫиЎЁиҝҮй•ҝдјҡиҪ¬еҢ–дёәзәўй»‘ж ‘гҖӮ
    • TreeMapеҹәдәҺзәўй»‘ж ‘е®һзҺ°пјҢжҳҜдёҖз§ҚиҮӘе№іиЎЎдәҢеҸүжҹҘжүҫж ‘пјҢиғҪеӨҹдҝқжҢҒй”®зҡ„жңүеәҸжҖ§пјҢеӣ жӯӨпјҢTreeMapдёӯзҡ„е…ғзҙ жҖ»жҳҜжҢүз…§й”®зҡ„иҮӘз„¶йЎәеәҸжҲ–иҖ…еҲӣе»әж—¶жҸҗдҫӣзҡ„ComparatorиҝӣиЎҢжҺ’еәҸгҖӮ
  • жҖ§иғҪзү№зӮ№пјҡ
    • HashMapзҡ„жҸ’е…ҘгҖҒеҲ йҷӨе’ҢжҹҘжүҫж“ҚдҪңзҡ„е№іеқҮж—¶й—ҙеӨҚжқӮеәҰдёәO(1)пјҢдҪҶеңЁжңҖеқҸжғ…еҶөдёӢпјҲдҫӢеҰӮеӨ§йҮҸе“ҲеёҢеҶІзӘҒпјүеҸҜиғҪйҖҖеҢ–дёәO(n)гҖӮ
    • TreeMapзҡ„жҸ’е…ҘгҖҒеҲ йҷӨе’ҢжҹҘжүҫж“ҚдҪңзҡ„ж—¶й—ҙеӨҚжқӮеәҰдёәO(log n)пјҢеӣ дёәе®ғйңҖиҰҒз»ҙжҠӨзәўй»‘ж ‘зҡ„е№іиЎЎжҖ§гҖӮ

4.HashSetе’ҢHashMapзҡ„еҢәеҲ«

  • еӯҳеӮЁеҶ…е®№дёҚеҗҢпјҡ
    • HashSetеӯҳеӮЁзҡ„жҳҜдёҚйҮҚеӨҚзҡ„еҚ•дёӘе…ғзҙ гҖӮе®ғдё»иҰҒз”ЁжқҘеҝ«йҖҹжЈҖжҹҘжҹҗдёӘе…ғзҙ жҳҜеҗҰеӯҳеңЁдәҺйӣҶеҗҲдёӯгҖӮ
    • HashMapеӯҳеӮЁзҡ„жҳҜй”®еҖјеҜ№пјҲkey-value pairsпјүгҖӮе®ғз”ЁдәҺж №жҚ®й”®еҝ«йҖҹжҹҘжүҫеҜ№еә”зҡ„еҖјгҖӮ
  • еҶ…йғЁе®һзҺ°е…ізі»пјҡ
    • HashSetе®һйҷ…дёҠжҳҜеҹәдәҺHashMapе®һзҺ°зҡ„гҖӮжҜҸеҪ“дҪ еҗ‘HashSetж·»еҠ дёҖдёӘе…ғзҙ ж—¶пјҢе®һйҷ…дёҠжҳҜе°ҶиҜҘе…ғзҙ дҪңдёәHashMapзҡ„й”®еӯҳеӮЁпјҢиҖҢеҖјеҲҷжҳҜдёҖдёӘеӣәе®ҡзҡ„еёёйҮҸеҜ№иұЎпјҲйҖҡеёёжҳҜдёҖдёӘз§Ғжңүзҡ„йқҷжҖҒеёёйҮҸпјүгҖӮ

5. HashMapе’ҢHashtableзҡ„еҢәеҲ«

  • зәҝзЁӢе®үе…ЁжҖ§пјҡ
    • HashMapжҳҜйқһзәҝзЁӢе®үе…Ёзҡ„пјҢеӨҡдёӘзәҝзЁӢеҗҢж—¶и®ҝй—®ж—¶еҸҜиғҪдјҡеҜјиҮҙж•°жҚ®дёҚдёҖиҮҙгҖӮеҰӮжһңйңҖиҰҒеңЁеӨҡзәҝзЁӢзҺҜеўғдёӢдҪҝз”ЁпјҢеҸҜд»ҘйҖҡиҝҮCollections.synchronizedMap()ж–№жі•е°Ҷе…¶еҢ…иЈ…дёәзәҝзЁӢе®үе…Ёзҡ„зүҲжң¬пјҢжҲ–иҖ…дҪҝз”ЁConcurrentHashMapгҖӮ
    • HashtableжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢжүҖжңүзҡ„ж–№жі•йғҪдҪҝз”Ёsynchronizedе…ій”®еӯ—иҝӣиЎҢеҗҢжӯҘпјҢзЎ®дҝқеңЁеӨҡзәҝзЁӢзҺҜеўғдёӢзҡ„е®үе…ЁжҖ§гҖӮдҪҶиҝҷз§ҚеҗҢжӯҘжңәеҲ¶дјҡеёҰжқҘжҖ§иғҪејҖй”ҖгҖӮ
  • nullй”®е’ҢnullеҖјпјҡ
    • HashMapе…Ғи®ёдёҖдёӘnullй”®е’ҢеӨҡдёӘnullеҖјгҖӮ
    • HashtableдёҚе…Ғи®ёnullй”®жҲ–nullеҖјпјҢе°қиҜ•жҸ’е…ҘnullдјҡжҠӣеҮәNullPointerExceptionгҖӮ
  • жҖ§иғҪпјҡ
    • HashMapз”ұдәҺйқһзәҝзЁӢе®үе…ЁпјҢеңЁеҚ•зәҝзЁӢзҺҜеўғдёӢйҖҡеёёжҜ”HashtableжҖ§иғҪжӣҙеҘҪгҖӮ
    • Hashtableз”ұдәҺеҗҢжӯҘжңәеҲ¶пјҢеңЁеӨҡзәҝзЁӢзҺҜеўғдёӢжҖ§иғҪиҫғе·®гҖӮеңЁйңҖиҰҒзәҝзЁӢе®үе…Ёзҡ„еңәжҷҜдёӢпјҢжӣҙжҺЁиҚҗдҪҝз”ЁConcurrentHashMapгҖӮ
  • зҲ¶зұ»пјҡ
    • HashMapз»§жүҝиҮӘAbstractMapпјҢе®һзҺ°дәҶMapжҺҘеҸЈгҖӮ
    • Hashtableз»§жүҝиҮӘDictionaryпјҢд№ҹжҳҜе®һзҺ°дәҶMapжҺҘеҸЈпјҢдҪҶDictionaryжҳҜдёҖдёӘиҫғж—©зҡ„зұ»пјҢе·Із»Ҹиў«йҒ—ејғгҖӮ
  • жҖ»з»“пјҡ
    • HashtableжҳҜдёҖдёӘйҒ—з•ҷзҡ„зәҝзЁӢе®үе…Ёзұ»пјҢжҖ§иғҪиҫғе·®гҖӮеҰӮжһңйңҖиҰҒзәҝзЁӢе®үе…ЁпјҢжҺЁиҚҗдҪҝз”ЁConcurrentHashMapгҖӮеҰӮжһңдёҚйңҖиҰҒзәҝзЁӢе®үе…ЁпјҢдҪҝз”ЁHashMapгҖӮ

6. е®һзҺ°HashMapзҡ„зәҝзЁӢе®үе…Ёзҡ„ж–№жі•

  • и°ғз”ЁCollections.syncronizedMap()ж–№жі•жқҘе®һзҺ°зәҝзЁӢе®үе…ЁпјҢдјҡеҜ№жүҖжңүж–№жі•и°ғз”ЁеҠ й”ҒпјҢзЎ®дҝқеҗҢдёҖж—¶еҲ»еҸӘжңүдёҖдёӘзәҝзЁӢиғҪеӨҹи®ҝй—®йӣҶеҗҲеҜ№иұЎгҖӮ
  • дҪҝз”ЁзәҝзЁӢе®үе…Ёзҡ„йӣҶеҗҲзұ»пјҢеҰӮConcurrentHashMapзұ»пјҢе°ұжҳҜзәҝзЁӢе®үе…Ёзҡ„

7. HashMapеңЁJDK1.8дёӯзҡ„ж”№еҠЁ

7.1 еј•е…Ҙзәўй»‘ж ‘

  • JDK1.7д№ӢеүҚпјҡ е“ҲеёҢеҶІзӘҒж—¶йҮҮз”ЁжӢүй“ҫжі•пјҲй“ҫиЎЁеӯҳеӮЁпјүпјҢеңЁй“ҫиЎЁиҝҮй•ҝж—¶жҹҘиҜўж•ҲзҺҮйҖҖеҢ–дёәO(n)гҖӮ
  • JDK1.8еҸҠд»ҘеҗҺпјҡ еҪ“еҚ•дёӘжЎ¶еҶ…зҡ„й“ҫиЎЁй•ҝеәҰи¶…иҝҮйҳҲеҖјпјҲй»ҳи®Ө8пјүпјҢе№¶дё”еҪ“еүҚHashMapзҡ„е®№йҮҸеӨ§дәҺ64ж—¶пјҢй“ҫиЎЁдјҡиҪ¬жҚўдёәзәўй»‘ж ‘пјҢд»ҺиҖҢе°ҶжҹҘиҜўж•ҲзҺҮжҸҗеҚҮеҲ°O(log n)гҖӮ
    • еҪ“е…ғзҙ еҮҸе°‘еҲ°6д»ҘдёӢж—¶пјҢзәўй»‘ж ‘дјҡйҖҖеҢ–еӣһй“ҫиЎЁгҖӮ
    • зәўй»‘ж ‘зҡ„еј•е…Ҙжҳҫи‘—жҸҗеҚҮдәҶеңЁй«ҳеҶІзӘҒжғ…еҶөдёӢзҡ„жҖ§иғҪгҖӮ

7.2 ж•°з»„+й“ҫиЎЁ/зәўй»‘ж ‘зҡ„ж··еҗҲз»“жһ„

  • JDK1.7д№ӢеүҚпјҡ ж•°з»„ + й“ҫиЎЁгҖӮ
  • JDK1.8еҸҠд»ҘеҗҺпјҡ ж•°з»„ + й“ҫиЎЁ + зәўй»‘ж ‘гҖӮ

7.3 дҪҝз”Ёе°ҫжҸ’жі•жӣҝд»ЈеӨҙжҸ’жі•

  • JDK1.7д№ӢеүҚпјҡ ж–°иҠӮзӮ№жҸ’е…Ҙж—¶йҮҮз”ЁеӨҙжҸ’жі•пјҢеҸҜиғҪеҜјиҮҙй“ҫиЎЁеҸҚиҪ¬пјҢеңЁеӨҡзәҝзЁӢзҺҜеўғдёӢе®№жҳ“еҮәзҺ°зҺҜеҪўй“ҫиЎЁ->жӯ»еҫӘзҺҜгҖӮ
  • JDK1.8еҸҠд»ҘеҗҺпјҡ ж”№з”Ёе°ҫжҸ’жі•пјҢдҝқжҢҒжҸ’е…ҘйЎәеәҸпјҢеҮҸе°‘е№¶еҸ‘й—®йўҳгҖӮ

7.4 жү©е®№жңәеҲ¶дјҳеҢ–

  • JDK1.7д№ӢеүҚпјҡ жү©е®№ж—¶йңҖиҰҒйҮҚж–°и®Ўз®—жҜҸдёӘе…ғзҙ зҡ„зҙўеј•дҪҚзҪ®пјҢејҖй”ҖиҫғеӨ§гҖӮ
  • JDK1.8еҸҠд»ҘеҗҺпјҡ еҲ©з”Ёй«ҳдҪҚиҝҗз®—пјҲе…ғзҙ иҰҒд№ҲеңЁеҺҹзҙўеј•пјҢиҰҒд№Ҳз§»еҠЁеҲ°еҺҹзҙўеј• + oldCapпјүпјҢеҸӘйңҖеҲӨж–ӯдёҖдёӘдәҢиҝӣеҲ¶дҪҚеҚіеҸҜпјҢеҮҸе°‘дәҶи®Ўз®—ејҖй”ҖгҖӮ

7.5 hashеҮҪж•°дјҳеҢ–

  • JDK1.7д№ӢеүҚпјҡ зӣҙжҺҘдҪҝз”Ёkeyзҡ„hashCode()й«ҳдҪҚеҸӮдёҺиҝҗз®—гҖӮ
  • JDK1.8еҸҠд»ҘеҗҺпјҡ еј•е…Ҙжү°еҠЁеҮҪж•°пјҢйҖҡиҝҮ(h = key.hashCode()) ^ (h >>> 16)е°Ҷй«ҳ16дҪҚе’ҢдҪҺ16дҪҚиҝӣиЎҢејӮжҲ–иҝҗз®—пјҢеҮҸе°‘е“ҲеёҢеҶІзӘҒпјҢжҸҗй«ҳеҲҶеёғеқҮеҢҖжҖ§гҖӮ

7.6 жҖ»з»“
