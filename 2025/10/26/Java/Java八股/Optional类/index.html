Optionalзұ»

1. з®ҖиҰҒеӣһзӯ”пјҡ

  • Optionalзұ»зҡ„е®ҡд№үпјҡ
    • еҢ…иЈ…зұ»пјҡ е°ҶеҸҜиғҪдёәnullзҡ„еҖје°ҒиЈ…жҲҗOptionalеҜ№иұЎпјҲеҰӮOptional<String>пјүгҖӮ
    • жҳҺзЎ®з©әеҖјж„Ҹеӣҫпјҡ йҖҡиҝҮж–№жі•еҗҚпјҲеҰӮisPresent()гҖҒorElse()пјүжҳҫејҸиЎЁиҫҫвҖңеҖјеҸҜиғҪдёҚеӯҳеңЁвҖқзҡ„йҖ»иҫ‘гҖӮ
  • Optionalзұ»зҡ„дҪңз”Ёпјҡ
    • йҒҝе…ҚNullPointerExceptionпјҡ ејәеҲ¶ејҖеҸ‘иҖ…дё»еҠЁеӨ„зҗҶз©әеҖјпјҢеҮҸе°‘иҝҗиЎҢж—¶й”ҷиҜҜгҖӮ
    • д»Јз Ғжӣҙз®ҖжҙҒпјҡ й“ҫејҸи°ғз”Ёжӣҝд»ЈеӨҡеұӮif (obj != null)еҲӨз©әгҖӮ
    • APIи®ҫи®Ўжӣҙжё…жҷ°пјҡ ж–№жі•зӯҫеҗҚдёӯжҳҺзЎ®иЎЁзӨәиҝ”еӣһеҖјеҸҜиғҪдёәз©әпјҲеҰӮOptional<User> findUser()пјүгҖӮ

2. дёәд»Җд№ҲйңҖиҰҒ Optionalпјҹ

  1. йҒҝе…Қз©әжҢҮй’ҲејӮеёёпјҡдј з»ҹзҡ„д»Јз ҒдёӯпјҢnull еҖјз»Ҹеёёиў«з”ЁжқҘиЎЁзӨәжҹҗдәӣеӯ—ж®өгҖҒиҝ”еӣһеҖјзӯүзјәеӨұзҡ„жғ…еҶөгҖӮеҰӮжһңдёҚе°ҸеҝғдҪҝз”ЁдәҶиҝҷдәӣеҖјпјҢе°ұдјҡжҠӣеҮә NullPointerExceptionгҖӮйҖҡиҝҮдҪҝз”Ё OptionalпјҢдҪ жҳҺзЎ®е‘ҠиҜүи°ғз”ЁиҖ…иҝҷдёӘеҖјеҸҜиғҪжҳҜз©әзҡ„пјҢд»ҺиҖҢйҷҚдҪҺдәҶеҮәй”ҷзҡ„жҰӮзҺҮгҖӮ

  2. еўһејәд»Јз ҒеҸҜиҜ»жҖ§е’ҢиЎЁиҫҫжҖ§пјҡOptional жҳҫзӨәең°иЎЁжҳҺдёҖдёӘеҖјеҸҜиғҪдёҚеӯҳеңЁжҲ–дёәз©әпјҢдҪҝеҫ—д»Јз ҒеңЁеӨ„зҗҶзјәеӨұеҖјж—¶жӣҙеҠ жё…жҷ°гҖҒеҸҜиҜ»гҖӮдј з»ҹзҡ„еҒҡжі•еҸҜиғҪдјҡз”Ё null еҖјжқҘиЎЁзӨәпјҢдҪҶиҝҷеҫҖеҫҖдёҚеӨҹзӣҙи§ӮгҖӮ

  3. жӣҙеҘҪзҡ„й“ҫејҸж“ҚдҪңпјҡOptional жҸҗдҫӣдәҶеҫҲеӨҡж–№жі•жқҘж“ҚдҪңеҖјпјҢдҫӢеҰӮ map(), flatMap(), filter(), ifPresent(), orElse() зӯүпјҢеҸҜд»ҘйҒҝе…ҚеҶ—й•ҝзҡ„ null жЈҖжҹҘе№¶е®һзҺ°еҮҪж•°ејҸзј–зЁӢйЈҺж јзҡ„й“ҫејҸи°ғз”ЁгҖӮ

3. Optional зҡ„еёёи§Ғж–№жі•пјҡ

  • Optional.of(T value)пјҡеҰӮжһң value жҳҜ nullпјҢдјҡжҠӣеҮә NullPointerExceptionгҖӮз”ЁдәҺеҢ…иЈ…йқһз©әеҖјгҖӮ
  • Optional.ofNullable(T value)пјҡеҰӮжһң value жҳҜ nullпјҢиҝ”еӣһдёҖдёӘз©әзҡ„ Optional е®һдҫӢпјӣеҗҰеҲҷпјҢиҝ”еӣһдёҖдёӘеҢ…еҗ«иҜҘеҖјзҡ„ OptionalгҖӮ
  • Optional.empty()пјҡиҝ”еӣһдёҖдёӘз©әзҡ„ Optional е®һдҫӢгҖӮ
  • Optional.isPresent()пјҡеҰӮжһңеҖјеӯҳеңЁпјҢиҝ”еӣһ trueпјӣеҗҰеҲҷпјҢиҝ”еӣһ falseгҖӮ
  • Optional.ifPresent(Consumer<? super T> action)пјҡеҰӮжһңеҖјеӯҳеңЁпјҢжү§иЎҢдј е…Ҙзҡ„ Consumer ж“ҚдҪңгҖӮ
  • Optional.orElse(T other)пјҡеҰӮжһңеҖјеӯҳеңЁпјҢиҝ”еӣһеҖјжң¬иә«пјӣеҰӮжһңеҖјдёәз©әпјҢиҝ”еӣһ otherгҖӮ
  • Optional.orElseGet(Supplier<? extends T> other)пјҡеҰӮжһңеҖјеӯҳеңЁпјҢиҝ”еӣһеҖјжң¬иә«пјӣеҰӮжһңеҖјдёәз©әпјҢи°ғз”ЁжҸҗдҫӣзҡ„ Supplier жқҘз”ҹжҲҗдёҖдёӘжӣҝд»ЈеҖјгҖӮ
  • Optional.orElseThrow(Supplier<? extends X> exceptionSupplier)пјҡеҰӮжһңеҖјеӯҳеңЁпјҢиҝ”еӣһеҖјжң¬иә«пјӣеҰӮжһңеҖјдёәз©әпјҢжҠӣеҮәз”ұ exceptionSupplier жҸҗдҫӣзҡ„ејӮеёёгҖӮ

4. зӨәдҫӢпјҡ

5. дҪҝз”Ё Optional зҡ„жіЁж„ҸдәӢйЎ№пјҡ

  1. йҒҝе…ҚиҝҮеәҰдҪҝз”ЁпјҡиҷҪз„¶ Optional еҫҲжңүз”ЁпјҢдҪҶе®ғе№¶дёҚйҖӮеҗҲз”ЁдәҺжүҖжңүеңәжҷҜгҖӮжҜ”еҰӮпјҢеңЁйӣҶеҗҲдёӯеӯҳеӮЁеӨҡдёӘе…ғзҙ ж—¶пјҢOptional дёҚжҳҜжңҖдҪійҖүжӢ©гҖӮеҸҰеӨ–пјҢOptional дё»иҰҒз”ЁдәҺиҝ”еӣһеҖјпјҢиҖҢдёҚеә”иҜҘз”ЁдәҺж–№жі•зҡ„еҸӮж•°зұ»еһӢгҖӮ

  2. жҖ§иғҪиҖғиҷ‘пјҡOptional дҪңдёәдёҖдёӘе®№еҷЁзұ»дјҡеўһеҠ дёҖдәӣжҖ§иғҪејҖй”ҖпјҢзү№еҲ«жҳҜеңЁж¶үеҸҠеӨ§йҮҸж•°жҚ®ж“ҚдҪңж—¶гҖӮе°Ҫз®ЎеҰӮжӯӨпјҢиҝҷз§ҚејҖй”ҖдёҖиҲ¬жқҘиҜҙжҳҜеҫ®д№Һе…¶еҫ®зҡ„гҖӮ

жҖ»з»“жқҘиҜҙпјҢOptional жҸҗдҫӣдәҶдёҖз§Қжӣҙе®үе…ЁгҖҒжҳ“иҜ»зҡ„ж–№ејҸжқҘеӨ„зҗҶеҸҜиғҪдёәз©әзҡ„еҖјпјҢд»ҺиҖҢйҒҝе…ҚдәҶз©әжҢҮй’ҲејӮеёёпјҢе№¶дҪҝеҫ—д»Јз Ғжӣҙе…·иЎЁиҫҫжҖ§гҖӮ
