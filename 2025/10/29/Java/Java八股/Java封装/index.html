  • жҸҗй«ҳе®үе…ЁжҖ§пјҡ йҖҡиҝҮжҺ§еҲ¶ж•°жҚ®зҡ„и®ҝй—®жқғйҷҗпјҢйҒҝе…ҚзӣҙжҺҘдҝ®ж”№еҶ…йғЁзҠ¶жҖҒпјҢзЎ®дҝқж•°жҚ®зҡ„е®Ңж•ҙжҖ§гҖӮ
  • з®ҖеҢ–еӨҚжқӮжҖ§пјҡ еӨ–йғЁеҸӘйңҖиҰҒе…іеҝғеҰӮдҪ•и°ғз”Ёж–№жі•пјҢиҖҢдёҚйңҖиҰҒдәҶи§ЈеҶ…йғЁзҡ„еӨҚжқӮе®һзҺ°гҖӮ

    • жҸҗдҫӣе…¬е…ұзҡ„getterе’Ңsetterж–№жі•пјҢз”ЁдәҺи®ҝй—®е’Ңдҝ®ж”№з§ҒжңүеұһжҖ§гҖӮ

  • зӨәдҫӢпјҡй“¶иЎҢиҙҰжҲ·пјҲе°ҒиЈ…зҡ„еә”з”Ёпјү

    • еӨ–йғЁд»Јз ҒдёҚиғҪзӣҙжҺҘдҝ®ж”№дҪҷйўқпјҢиҖҢжҳҜеҝ…йЎ»йҖҡиҝҮзұ»жҸҗдҫӣзҡ„depositе’Ңwithdrawж–№жі•жқҘж“ҚдҪңгҖӮ
