JavaеӨҡзәҝзЁӢ

1. дёәд»Җд№ҲдҪҝз”ЁеӨҡзәҝзЁӢ

  • жҖ»дҪ“дёҠиҜҙпјҡ
    • д»Һи®Ўз®—жңәеә•еұӮжқҘиҜҙпјҡзәҝзЁӢеҸҜд»ҘжҜ”дҪңжҳҜиҪ»йҮҸзә§зҡ„иҝӣзЁӢпјҢжҳҜзЁӢеәҸжү§иЎҢзҡ„жңҖе°ҸеҚ•дҪҚпјҢзәҝзЁӢй—ҙзҡ„еҲҮжҚўе’Ңи°ғеәҰжҲҗжң¬иҝңдҪҺдәҺиҝӣзЁӢгҖӮдё”еӨҡж ёCPUеҸҜд»ҘеҮҸе°‘зәҝзЁӢдёҠдёӢж–ҮеҲҮжҚўзҡ„ејҖй”Җ
    • д»Һдә’иҒ”зҪ‘еҸ‘еұ•и¶ӢеҠҝжқҘиҜҙпјҡеӨҡзәҝзЁӢе№¶еҸ‘зј–зЁӢжҳҜејҖеҸ‘й«ҳе№¶еҸ‘зі»з»ҹзҡ„еҹәзЎҖ
  • ж·ұе…ҘеҲ°и®Ўз®—жңәеә•еұӮи®Ёи®әпјҡ
    • еҚ•ж ёж—¶д»ЈпјҡеӨҡзәҝзЁӢдё»иҰҒжҳҜдёәдәҶжҸҗй«ҳеҚ•иҝӣзЁӢзҡ„еҲ©з”ЁCPUе’ҢIOзі»з»ҹзҡ„ж•ҲзҺҮ
    • еӨҡж ёж—¶д»ЈпјҡжҸҗй«ҳиҝӣзЁӢеҲ©з”ЁеӨҡж ёCPUзҡ„иғҪеҠӣ

2. еҚ•ж ёCPUж”ҜжҢҒеӨҡзәҝзЁӢеҗ—

  • ж”ҜжҢҒпјҢдҪҝз”Ёж—¶й—ҙзүҮиҪ®иҪ¬зҡ„ж–№ејҸе®һзҺ°еӨҡзәҝзЁӢ
  • ж“ҚдҪңзі»з»ҹдё»иҰҒжҳҜйҖҡиҝҮдёӨз§ҚзәҝзЁӢи°ғеәҰж–№ејҸжқҘз®ЎзҗҶеӨҡзәҝзЁӢжү§иЎҢпјҡ
    • жҠўеҚ ејҸи°ғеәҰпјҲPreemptive Schedulingпјүпјҡж“ҚдҪңзі»з»ҹеҶіе®ҡдҪ•ж—¶жҡӮеҒңеҪ“еүҚжӯЈеңЁиҝҗиЎҢзҡ„зәҝзЁӢпјҢе№¶еҲҮжҚўеҲ°еҸҰдёҖдёӘзәҝзЁӢжү§иЎҢгҖӮиҝҷз§ҚеҲҮжҚўйҖҡеёёжҳҜз”ұ**зі»з»ҹж—¶й’ҹдёӯж–ӯпјҲж—¶й—ҙзүҮиҪ®иҪ¬пјүжҲ–е…¶д»–й«ҳдјҳе…Ҳзә§дәӢд»¶пјҲеҰӮ I/O ж“ҚдҪңе®ҢжҲҗпјү**и§ҰеҸ‘зҡ„гҖӮиҝҷз§Қж–№ејҸеӯҳеңЁдёҠдёӢж–ҮеҲҮжҚўејҖй”ҖпјҢдҪҶе…¬е№іжҖ§е’Ң CPU иө„жәҗеҲ©з”ЁзҺҮиҫғеҘҪпјҢдёҚжҳ“йҳ»еЎһгҖӮ
    • еҚҸеҗҢејҸи°ғеәҰпјҲCooperative SchedulingпјүпјҡзәҝзЁӢжү§иЎҢе®ҢжҜ•еҗҺпјҢдё»еҠЁйҖҡзҹҘзі»з»ҹеҲҮжҚўеҲ°еҸҰдёҖдёӘзәҝзЁӢгҖӮиҝҷз§Қж–№ејҸеҸҜд»ҘеҮҸе°‘дёҠдёӢж–ҮеҲҮжҚўеёҰжқҘзҡ„жҖ§иғҪејҖй”ҖпјҢдҪҶе…¬е№іжҖ§иҫғе·®пјҢе®№жҳ“йҳ»еЎһгҖӮ
  • JavaдҪҝз”Ёзҡ„зәҝзЁӢи°ғеәҰжҳҜжҠўеҚ ејҸзҡ„пјҢJVMжң¬иә«дёҚиҙҹиҙЈзәҝзЁӢзҡ„и°ғеәҰпјҢиҖҢжҳҜе°ҶзәҝзЁӢзҡ„и°ғеәҰе§”жүҳз»ҷж“ҚдҪңзі»з»ҹгҖӮж“ҚдҪңзі»з»ҹйҖҡеёёдјҡеҹәдәҺзәҝзЁӢдјҳе…Ҳзә§е’Ңж—¶й—ҙзүҮжқҘи°ғеәҰзәҝзЁӢзҡ„жү§иЎҢпјҢй«ҳдјҳе…Ҳзә§зҡ„зәҝзЁӢйҖҡеёёиҺ·еҫ—CPUж—¶й—ҙзүҮзҡ„жңәдјҡжӣҙеӨҡ

3. еҚ•ж ёCPUдёҠиҝҗиЎҢеӨҡдёӘзәҝзЁӢж•ҲзҺҮдёҖе®ҡдјҡй«ҳеҗ—

  • дёҚдёҖе®ҡпјҢеҸ–еҶідәҺзәҝзЁӢзҡ„зұ»еһӢе’Ңд»»еҠЎзҡ„жҖ§иҙЁ
  • дёҖиҲ¬жқҘиҜҙпјҢжңүдёӨз§Қзұ»еһӢзҡ„зәҝзЁӢпјҡ
    • CPUеҜҶйӣҶеһӢпјҡдё»иҰҒиҝӣиЎҢи®Ўз®—е’ҢйҖ»иҫ‘еӨ„зҗҶпјҢйңҖиҰҒеҚ з”ЁеӨ§йҮҸзҡ„CPUиө„жәҗ
    • IOеҜҶйӣҶеһӢпјҡдё»иҰҒиҝӣиЎҢIOж“ҚдҪңпјҢеҰӮж–Үд»¶иҜ»еҶҷгҖҒзҪ‘з»ңйҖҡдҝЎзӯүпјҢCPUиө„жәҗеҚ з”Ёиҫғе°‘
  • еңЁеҚ•ж ёCPUдёҠиҝҗиЎҢеӨҡдёӘзәҝзЁӢж—¶пјҡ
    • еҜ№дәҺCPUеҜҶйӣҶеһӢд»»еҠЎпјҡеӨҡдёӘзәҝзЁӢдјҡдәүеӨәCPUиө„жәҗпјҢйў‘з№Ғзҡ„зәҝзЁӢеҲҮжҚўдјҡеёҰжқҘиҫғеӨ§зҡ„дёҠдёӢж–ҮеҲҮжҚўејҖй”ҖпјҢеҸҚиҖҢеҸҜиғҪйҷҚдҪҺж•ҙдҪ“ж•ҲзҺҮ
    • еҜ№дәҺIOеҜҶйӣҶеһӢд»»еҠЎпјҡзәҝзЁӢеңЁзӯүеҫ…IOж“ҚдҪңе®ҢжҲҗж—¶дјҡйҮҠж”ҫCPUиө„жәҗпјҢе…¶д»–зәҝзЁӢеҸҜд»ҘеҲ©з”Ёиҝҷдәӣз©әй—Іж—¶й—ҙжү§иЎҢд»»еҠЎпјҢд»ҺиҖҢжҸҗй«ҳж•ҙдҪ“ж•ҲзҺҮ
