Leetcode 452.з”ЁжңҖе°‘ж•°йҮҸзҡ„з®ӯеј•зҲҶж°”зҗғ

йўҳзӣ®иҰҒжұӮ

  • жңүдёҖдәӣзҗғеҪўж°”зҗғиҙҙеңЁдёҖе өз”Ё XY е№ійқўиЎЁзӨәзҡ„еўҷйқўдёҠгҖӮеўҷйқўдёҠзҡ„ж°”зҗғи®°еҪ•еңЁж•ҙж•°ж•°з»„ points пјҢе…¶дёӯpoints[i] = [xstart, xend] иЎЁзӨәж°ҙе№ізӣҙеҫ„еңЁ xstart е’Ң xendд№Ӣй—ҙзҡ„ж°”зҗғгҖӮдҪ дёҚзҹҘйҒ“ж°”зҗғзҡ„зЎ®еҲҮ y еқҗж ҮгҖӮ

  • дёҖж”Ҝеј“з®ӯеҸҜд»ҘжІҝзқҖ x иҪҙд»ҺдёҚеҗҢзӮ№ е®Ңе…ЁеһӮзӣҙ ең°е°„еҮәгҖӮеңЁеқҗж Ү x еӨ„е°„еҮәдёҖж”Ҝз®ӯпјҢиӢҘжңүдёҖдёӘж°”зҗғзҡ„зӣҙеҫ„зҡ„ејҖе§Ӣе’Ңз»“жқҹеқҗж Үдёә xstartпјҢxendпјҢ дё”ж»Ўи¶і xstart вүӨ x вүӨ xendпјҢеҲҷиҜҘж°”зҗғдјҡиў« еј•зҲҶ гҖӮеҸҜд»Ҙе°„еҮәзҡ„еј“з®ӯзҡ„ж•°йҮҸ жІЎжңүйҷҗеҲ¶ гҖӮ еј“з®ӯдёҖж—Ұиў«е°„еҮәд№ӢеҗҺпјҢеҸҜд»Ҙж— йҷҗең°еүҚиҝӣгҖӮ

  • з»ҷдҪ дёҖдёӘж•°з»„ points пјҢиҝ”еӣһеј•зҲҶжүҖжңүж°”зҗғжүҖеҝ…йЎ»е°„еҮәзҡ„ жңҖе°Ҹ еј“з®ӯж•° гҖӮ

зӨәдҫӢ 1пјҡ
иҫ“е…Ҙпјҡpoints = [[10,16],[2,8],[1,6],[7,12]]
иҫ“еҮәпјҡ2
и§ЈйҮҠпјҡж°”зҗғеҸҜд»Ҙз”Ё2ж”Ҝз®ӯжқҘзҲҶз ҙ:
-еңЁx = 6еӨ„е°„еҮәз®ӯпјҢеҮ»з ҙж°”зҗғ[2,8]е’Ң[1,6]гҖӮ
-еңЁx = 11еӨ„еҸ‘е°„з®ӯпјҢеҮ»з ҙж°”зҗғ[10,16]е’Ң[7,12]гҖӮ

зӨәдҫӢ 2:
иҫ“е…Ҙпјҡpoints = [[1,2],[3,4],[5,6],[7,8]]
иҫ“еҮәпјҡ4
и§ЈйҮҠпјҡжҜҸдёӘж°”зҗғйңҖиҰҒе°„еҮәдёҖж”Ҝз®ӯпјҢжҖ»е…ұйңҖиҰҒ4ж”Ҝз®ӯгҖӮ

зӨәдҫӢ 3пјҡ
иҫ“е…Ҙпјҡpoints = [[1,2],[2,3],[3,4],[4,5]]
иҫ“еҮәпјҡ2
и§ЈйҮҠпјҡж°”зҗғеҸҜд»Ҙз”Ё2ж”Ҝз®ӯжқҘзҲҶз ҙ:

  • еңЁx = 2еӨ„еҸ‘е°„з®ӯпјҢеҮ»з ҙж°”зҗғ[1,2]е’Ң[2,3]гҖӮ
  • еңЁx = 4еӨ„е°„еҮәз®ӯпјҢеҮ»з ҙж°”зҗғ[3,4]е’Ң[4,5]гҖӮ

иҙӘеҝғ

жҖқи·Ҝпјҡ
йҰ–е…ҲжҢүз…§е·Ұиҫ№з•ҢиҝӣиЎҢжҺ’еәҸ
жңҖеӨҡйңҖиҰҒpoints.lengthж”Ҝз®ӯпјҢжҜҸжңүдёҖдёӘйҮҚеҸ зҡ„еҢәй—ҙе°ұеҸҜд»ҘеҮҸе°‘дёҖж”Ҝз®ӯ

д»Јз ҒйҡҸжғіеҪ•

зӣҙи§үдёҠжқҘзңӢпјҢиІҢдјјеҸӘе°„йҮҚеҸ жңҖеӨҡзҡ„ж°”зҗғпјҢз”Ёзҡ„еј“з®ӯдёҖе®ҡжңҖе°‘пјҢйӮЈд№ҲжңүжІЎжңүеҪ“еүҚйҮҚеҸ дәҶдёүдёӘж°”зҗғпјҢжҲ‘е°„дёӨдёӘпјҢз•ҷдёӢдёҖдёӘе’ҢеҗҺйқўзҡ„дёҖиө·е°„иҝҷж ·еј“з®ӯз”Ёзҡ„жӣҙе°‘зҡ„жғ…еҶөе‘ўпјҹ

е°қиҜ•дёҖдёӢдёҫеҸҚдҫӢпјҢеҸ‘зҺ°жІЎжңүиҝҷз§Қжғ…еҶөгҖӮ

йӮЈд№Ҳе°ұиҜ•дёҖиҜ•иҙӘеҝғеҗ§пјҒеұҖйғЁжңҖдјҳпјҡеҪ“ж°”зҗғеҮәзҺ°йҮҚеҸ пјҢдёҖиө·е°„пјҢжүҖз”Ёеј“з®ӯжңҖе°‘гҖӮе…ЁеұҖжңҖдјҳпјҡжҠҠжүҖжңүж°”зҗғе°„зҲҶжүҖз”Ёеј“з®ӯжңҖе°‘гҖӮ

з®—жі•зЎ®е®ҡдёӢжқҘдәҶпјҢйӮЈд№ҲеҰӮдҪ•жЁЎжӢҹж°”зҗғе°„зҲҶзҡ„иҝҮзЁӢе‘ўпјҹжҳҜеңЁж•°з»„дёӯз§»йҷӨе…ғзҙ иҝҳжҳҜеҒҡж Үи®°е‘ўпјҹ

еҰӮжһңзңҹе®һзҡ„жЁЎжӢҹе°„ж°”зҗғзҡ„иҝҮзЁӢпјҢеә”иҜҘе°„дёҖдёӘпјҢж°”зҗғж•°з»„е°ұremoveдёҖдёӘе…ғзҙ пјҢиҝҷж ·жңҖзӣҙи§ӮпјҢжҜ•з«ҹж°”зҗғиў«е°„дәҶгҖӮ

дҪҶд»”з»ҶжҖқиҖғдёҖдёӢе°ұеҸ‘зҺ°пјҡеҰӮжһңжҠҠж°”зҗғжҺ’еәҸд№ӢеҗҺпјҢд»ҺеүҚеҲ°еҗҺйҒҚеҺҶж°”зҗғпјҢиў«е°„иҝҮзҡ„ж°”зҗғд»…д»…и·іиҝҮе°ұиЎҢдәҶпјҢжІЎжңүеҝ…иҰҒи®©ж°”зҗғж•°з»„removeж°”зҗғпјҢеҸӘиҰҒи®°еҪ•дёҖдёӢз®ӯзҡ„ж•°йҮҸе°ұеҸҜд»ҘдәҶгҖӮ

д»ҘдёҠдёәжҖқиҖғиҝҮзЁӢпјҢе·Із»ҸзЎ®е®ҡдёӢжқҘдҪҝз”ЁиҙӘеҝғдәҶпјҢйӮЈд№ҲејҖе§Ӣи§ЈйўҳгҖӮ

дёәдәҶи®©ж°”зҗғе°ҪеҸҜиғҪзҡ„йҮҚеҸ пјҢйңҖиҰҒеҜ№ж•°з»„иҝӣиЎҢжҺ’еәҸгҖӮ

йӮЈд№ҲжҢүз…§ж°”зҗғиө·е§ӢдҪҚзҪ®жҺ’еәҸпјҢиҝҳжҳҜжҢүз…§ж°”зҗғз»ҲжӯўдҪҚзҪ®жҺ’еәҸе‘ўпјҹ

е…¶е®һйғҪеҸҜд»ҘпјҒеҸӘдёҚиҝҮеҜ№еә”зҡ„йҒҚеҺҶйЎәеәҸдёҚеҗҢпјҢжҲ‘е°ұжҢүз…§ж°”зҗғзҡ„иө·е§ӢдҪҚзҪ®жҺ’еәҸдәҶгҖӮ

ж—ўз„¶жҢүз…§иө·е§ӢдҪҚзҪ®жҺ’еәҸпјҢйӮЈд№Ҳе°ұд»ҺеүҚеҗ‘еҗҺйҒҚеҺҶж°”зҗғж•°з»„пјҢйқ е·Ұе°ҪеҸҜиғҪи®©ж°”зҗғйҮҚеӨҚгҖӮ

д»ҺеүҚеҗ‘еҗҺйҒҚеҺҶйҒҮеҲ°йҮҚеҸ зҡ„ж°”зҗғдәҶжҖҺд№ҲеҠһпјҹ

еҰӮжһңж°”зҗғйҮҚеҸ дәҶпјҢйҮҚеҸ ж°”зҗғдёӯеҸіиҫ№иҫ№з•Ңзҡ„жңҖе°ҸеҖј д№ӢеүҚзҡ„еҢәй—ҙдёҖе®ҡйңҖиҰҒдёҖдёӘеј“з®ӯгҖӮ

д»Ҙйўҳзӣ®зӨәдҫӢпјҡ [[10,16],[2,8],[1,6],[7,12]]дёәдҫӢпјҢеҰӮеӣҫпјҡпјҲж–№дҫҝиө·и§ҒпјҢе·Із»ҸжҺ’еәҸпјү


еҸҜд»ҘзңӢеҮәйҰ–е…Ҳз¬¬дёҖз»„йҮҚеҸ ж°”зҗғпјҢдёҖе®ҡжҳҜйңҖиҰҒдёҖдёӘз®ӯпјҢж°”зҗғ3пјҢзҡ„е·Ұиҫ№з•ҢеӨ§дәҺдәҶ з¬¬дёҖз»„йҮҚеҸ ж°”зҗғзҡ„жңҖе°ҸеҸіиҫ№з•ҢпјҢжүҖд»ҘеҶҚйңҖиҰҒдёҖж”Ҝз®ӯжқҘе°„ж°”зҗғ3дәҶгҖӮ
