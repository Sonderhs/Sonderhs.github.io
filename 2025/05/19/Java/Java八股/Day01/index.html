Day01 八股文答案（Java 基础）

1. Java 中值传递和引用传递的区别？

Java 只有值传递。对象作为参数传递时，传递的是引用的副本，即对象地址的值。修改对象内部属性会生效，但如果改变引用本身，不会影响原对象。

2. JDK 和 JRE 有什么区别？

  • JDK（Java Development Kit）：Java 开发工具包，包含 JRE + 编译器（javac）等工具。
  • JRE（Java Runtime Environment）：Java 运行环境，只包含 JVM 和类库，用于运行 Java 程序。

3. 面向对象的三大特性是什么？

  • 封装：隐藏内部实现细节，通过接口与外界交互。
  • 继承：子类继承父类，代码复用。
  • 多态：同一方法调用，不同表现形式（方法重写和接口实现）。

4. equals() 和 == 的区别？

  • == 比较的是两个引用是否指向同一个内存地址。
  • equals() 默认与 == 一样，但在类中可以重写，用于比较内容是否相等（如 String）。

5. String、StringBuilder 和 StringBuffer 区别？

  • String：不可变，操作会生成新对象。
  • StringBuilder：可变，线程不安全，效率高。
  • StringBuffer：可变，线程安全，效率比 StringBuilder 低。

6. 自动装箱和拆箱是什么？

Java 自动将基本类型与包装类相互转换：

  • 装箱：int → Integer。
  • 拆箱：Integer → int。

7. final、finally 和 finalize 区别？

  • final：修饰变量、方法、类，表示不可更改。
  • finally：异常处理机制中的代码块，无论是否异常都会执行。
  • finalize：对象被垃圾回收前由 GC 调用的方法（不推荐使用）。

8. HashMap 的底层实现原理？

底层是数组 + 链表 + 红黑树：

  • JDK1.7：数组 + 链表。
  • JDK1.8：链表长度 >8 时转换为红黑树，提高查找效率。

9. List 和 Set 的区别？

  • List：有序、可重复。
  • Set：无序、不可重复（底层使用哈希或树结构）。

10. Java 中如何实现多线程？

  1. 继承 Thread 类并重写 run() 方法。
  2. 实现 Runnable 接口并重写 run() 方法。
  3. 实现 Callable 接口并结合 FutureTask。
