  • иҜ·дҪ йҮҚж–°жһ„йҖ е№¶иҝ”еӣһиҫ“е…Ҙж•°з»„ people жүҖиЎЁзӨәзҡ„йҳҹеҲ—гҖӮиҝ”еӣһзҡ„йҳҹеҲ—еә”иҜҘж јејҸеҢ–дёәж•°з»„ queue пјҢе…¶дёӯ queue[j] = [hj, kj] жҳҜйҳҹеҲ—дёӯз¬¬ j дёӘдәәзҡ„еұһжҖ§пјҲqueue[0] жҳҜжҺ’еңЁйҳҹеҲ—еүҚйқўзҡ„дәәпјүгҖӮ
