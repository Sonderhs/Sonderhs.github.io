Java еӨҡзәҝзЁӢ&JUC

дёҖгҖҒеӨҡзәҝзЁӢжҰӮиҝ°

1.1 еӨҡзәҝзЁӢзҡ„жҰӮеҝө

  • зәҝзЁӢпјҡзәҝзЁӢжҳҜж“ҚдҪңзі»з»ҹиғҪеӨҹиҝӣиЎҢиҝҗз®—и°ғеәҰзҡ„жңҖе°ҸеҚ•дҪҚгҖӮе®ғиў«еҢ…еҗ«еңЁиҝӣзЁӢд№ӢдёӯпјҢдҪҝиҝӣзЁӢдёӯзҡ„е®һйҷ…иҝҗдҪңеҚ•дҪҚ
  • иҝӣзЁӢпјҡиҝӣзЁӢжҳҜзЁӢеәҸзҡ„еҹәжң¬жү§иЎҢе®һдҪ“
  • еӨҡзәҝзЁӢпјҡеӨҡзәҝзЁӢжҳҜжҢҮеңЁеҗҢдёҖиҝӣзЁӢдёӯе№¶еҸ‘жү§иЎҢеӨҡдёӘзәҝзЁӢзҡ„жҠҖжңҜгҖӮеӨҡзәҝзЁӢеҸҜд»ҘжҸҗй«ҳзЁӢеәҸзҡ„жү§иЎҢж•ҲзҺҮе’Ңе“Қеә”иғҪеҠӣ
  • еӨҡзәҝзЁӢзҡ„еә”з”ЁеңәжҷҜ
    • иҪҜд»¶дёӯзҡ„иҖ—ж—¶ж“ҚдҪң
      • жӢ·иҙқгҖҒиҝҒз§»еӨ§ж–Үд»¶
      • еҠ иҪҪеӨ§йҮҸзҡ„иө„жәҗж–Үд»¶
    • жүҖжңүзҡ„иҒҠеӨ©иҪҜд»¶
    • жүҖжңүзҡ„еҗҺеҸ°жңҚеҠЎеҷЁ

1.2 е№¶еҸ‘е’Ңе№¶иЎҢ

  • е№¶еҸ‘пјҡеңЁеҗҢдёҖж—¶еҲ»пјҢжңүеӨҡдёӘжҢҮд»ӨеңЁеҚ•дёӘCPUдёҠдәӨжӣҝжү§иЎҢ
  • е№¶иЎҢпјҡеңЁеҗҢдёҖж—¶еҲ»пјҢжңүеӨҡдёӘжҢҮд»ӨеңЁеӨҡдёӘCPUдёҠеҗҢж—¶жү§иЎҢ

дәҢгҖҒеӨҡзәҝзЁӢзҡ„е®һзҺ°ж–№ејҸ

2.1 з»§жүҝThreadзұ»

  • з»§жүҝThreadзұ»зҡ„ж–№ејҸе®һзҺ°еӨҡзәҝзЁӢ
  • жӯҘйӘӨпјҡ
    1. иҮӘе·ұе®ҡд№үдёҖдёӘзұ»з»§жүҝThread
    2. йҮҚеҶҷrunж–№жі•
    3. еҲӣе»әеӯҗзұ»зҡ„еҜ№иұЎпјҢе№¶еҗҜеҠЁзәҝзЁӢ

зӨәдҫӢпјҡ

2.2 е®һзҺ°RunnableжҺҘеҸЈ

  • е®һзҺ°RunnableжҺҘеҸЈзҡ„ж–№ејҸе®һзҺ°еӨҡзәҝзЁӢ
  • жӯҘйӘӨпјҡ
    1. иҮӘе·ұе®ҡд№үдёҖдёӘзұ»е®һзҺ°RunnableжҺҘеҸЈ
    2. йҮҚеҶҷrunж–№жі•
    3. еҲӣе»әRunnableжҺҘеҸЈзҡ„е®һзҺ°зұ»еҜ№иұЎпјҢе№¶е°Ҷе…¶дҪңдёәеҸӮж•°дј йҖ’з»ҷThreadзұ»зҡ„жһ„йҖ ж–№жі•
    4. и°ғз”ЁThreadзұ»зҡ„startж–№жі•еҗҜеҠЁзәҝзЁӢ

зӨәдҫӢпјҡ

2.3 CallableжҺҘеҸЈе’ҢFutureжҺҘеҸЈ

  • CallableжҺҘеҸЈе’ҢFutureжҺҘеҸЈзҡ„ж–№ејҸе®һзҺ°еӨҡзәҝзЁӢ
  • зү№зӮ№пјҡ
    • CallableжҺҘеҸЈеҸҜд»Ҙжңүиҝ”еӣһеҖјпјҢиҖҢRunnableжҺҘеҸЈдёҚиғҪжңүиҝ”еӣһеҖј
    • CallableжҺҘеҸЈеҸҜд»ҘжҠӣеҮәејӮеёёпјҢиҖҢRunnableжҺҘеҸЈдёҚиғҪжҠӣеҮәејӮеёё
  • жӯҘйӘӨпјҡ
    1. иҮӘе·ұе®ҡд№үдёҖдёӘзұ»MyCallableе®һзҺ°CallableжҺҘеҸЈ
    2. йҮҚеҶҷcallж–№жі•(жңүиҝ”еӣһеҖјпјҢиЎЁзӨәеӨҡзәҝзЁӢиҝҗиЎҢзҡ„з»“жһң)
    3. еҲӣе»әCallableжҺҘеҸЈзҡ„е®һзҺ°зұ»еҜ№иұЎ(иЎЁзӨәеӨҡзәҝзЁӢиҰҒжү§иЎҢзҡ„д»»еҠЎ)
    4. еҲӣе»әFutureTaskеҜ№иұЎ(з”ЁдәҺз®ЎзҗҶеӨҡзәҝзЁӢиҝҗиЎҢзҡ„з»“жһң)
    5. еҲӣе»әThreadеҜ№иұЎпјҢе№¶еҗҜеҠЁзәҝзЁӢ

зӨәдҫӢпјҡ

2.4 дёүз§Қе®һзҺ°ж–№ејҸеҜ№жҜ”

дјҳзӮ№ зјәзӮ№
з»§жүҝThreadзұ» зј–зЁӢжҜ”иҫғз®ҖеҚ•пјҢеҸҜд»ҘзӣҙжҺҘдҪҝз”ЁThreadзұ»дёӯзҡ„ж–№жі• еҸҜд»Ҙжү©еұ•жҖ§иҫғе·®пјҢдёҚиғҪеҶҚз»§жүҝе…¶д»–зұ»
е®һзҺ°RunnableжҺҘеҸЈ жү©еұ•жҖ§ејәпјҢе®һзҺ°иҜҘжҺҘеҸЈзҡ„еҗҢж—¶иҝҳеҸҜд»Ҙз»§жүҝе…¶д»–зұ» зј–зЁӢзӣёеҜ№еӨҚжқӮпјҢдёҚиғҪзӣҙжҺҘдҪҝз”ЁThreadзұ»дёӯзҡ„ж–№жі•
е®һзҺ°CallableжҺҘеҸЈ жү©еұ•жҖ§ејәпјҢе®һзҺ°иҜҘжҺҘеҸЈзҡ„еҗҢж—¶иҝҳеҸҜд»Ҙз»§жүҝе…¶д»–зұ» зј–зЁӢзӣёеҜ№еӨҚжқӮпјҢдёҚиғҪзӣҙжҺҘдҪҝз”ЁThreadзұ»дёӯзҡ„ж–№жі•

дёүгҖҒеёёи§Ғзҡ„жҲҗе‘ҳж–№жі•

  • String getName()пјҡиҝ”еӣһжӯӨзәҝзЁӢзҡ„еҗҚз§°
  • void setName(String name)пјҡи®ҫзҪ®жӯӨзәҝзЁӢзҡ„еҗҚз§°
  • static Thread currentThread()пјҡиҝ”еӣһеҜ№еҪ“еүҚжӯЈеңЁжү§иЎҢзҡ„зәҝзЁӢеҜ№иұЎ
  • static void sleep(long millis)пјҡдҪҝеҪ“еүҚзәҝзЁӢдј‘зң жҢҮе®ҡзҡ„жҜ«з§’ж•°
  • setPriority(int newPriority)пјҡи®ҫзҪ®зәҝзЁӢзҡ„дјҳе…Ҳзә§
  • final int getPriority()пјҡиҝ”еӣһзәҝзЁӢзҡ„дјҳе…Ҳзә§
  • final void setDaemon(boolean on)пјҡе°ҶзәҝзЁӢи®ҫзҪ®дёәе®ҲжҠӨзәҝзЁӢ
  • public static void yield()пјҡеҮәи®©зәҝзЁӢ/зӨји®©зәҝзЁӢ
  • public static void join()пјҡжҸ’е…ҘзәҝзЁӢ/жҸ’йҳҹзәҝзЁӢ

3.1 getName()гҖҒsetName()гҖҒcurrentThread()е’Ңsleep()

  • String getName()пјҡиҝ”еӣһжӯӨзәҝзЁӢзҡ„еҗҚз§°
    • еҰӮжһңжІЎжңүи®ҫзҪ®еҗҚеӯ—пјҢй»ҳи®ӨеҗҚеӯ—дёәThread-XпјҢXдёәеәҸеҸ·пјҢд»Һ0ејҖе§Ӣ
  • void setName(String name)пјҡи®ҫзҪ®жӯӨзәҝзЁӢзҡ„еҗҚз§°
    • зәҝзЁӢеҗҚз§°д№ҹеҸҜд»ҘдҪҝз”Ёжһ„йҖ ж–№жі•и®ҫзҪ®
    • дҪҝз”Ёжһ„йҖ ж–№жі•ж—¶йңҖиҰҒеңЁеӯҗзұ»дёӯз»§жүҝзҲ¶зұ»зҡ„жһ„йҖ ж–№жі•
  • static Thread currentThread()пјҡиҝ”еӣһеҜ№еҪ“еүҚжӯЈеңЁжү§иЎҢзҡ„зәҝзЁӢеҜ№иұЎ
    • еҪ“JVMиҷҡжӢҹжңәеҗҜеҠЁд№ӢеҗҺдјҡиҮӘеҠЁеҗҜеҠЁеӨҡжқЎзәҝзЁӢпјҢе…¶дёӯжңүдёҖжқЎзәҝзЁӢеҸ«еҒҡmainзәҝзЁӢ
    • mainзәҝзЁӢдҪңз”Ёе°ұжҳҜи°ғз”Ёmainж–№жі•пјҢе№¶жү§иЎҢйҮҢйқўзҡ„д»Јз Ғ
  • static void sleep(long millis)пјҡдҪҝеҪ“еүҚзәҝзЁӢдј‘зң жҢҮе®ҡзҡ„жҜ«з§’ж•°
    • е“ӘжқЎзәҝзЁӢжү§иЎҢеҲ°иҝҷдёӘж–№жі•пјҢе°ұдјҡеңЁиҝҷйҮҢеҒңз•ҷеҜ№еә”зҡ„ж—¶й—ҙ
    • ж—¶й—ҙеҲ°дәҶд№ӢеҗҺзәҝзЁӢдјҡиҮӘеҠЁйҶ’жқҘпјҢз»§з»ӯжү§иЎҢеҗҺйқўзҡ„д»Јз Ғ

зӨәдҫӢпјҡ

3.2 setPriority()е’ҢgetPriority()

  • жҠўеҚ ејҸи°ғеәҰпјҡзәҝзЁӢзҡ„дјҳе…Ҳзә§и¶Ҡй«ҳпјҢи¶ҠжңүеҸҜиғҪиҺ·еҫ—CPUзҡ„дҪҝз”Ёжқғ
  • йқһжҠўеҚ ејҸи°ғеәҰпјҡзәҝзЁӢзҡ„дјҳе…Ҳзә§и¶Ҡй«ҳпјҢи¶ҠжңүеҸҜиғҪиҺ·еҫ—CPUзҡ„дҪҝз”ЁжқғпјҢдҪҶдёҚдёҖе®ҡдјҡиҺ·еҫ—CPUзҡ„дҪҝз”Ёжқғ
  • дјҳе…Ҳзә§иҢғеӣҙпјҡ1-10
    • 1пјҡMIN_PRIORITYпјҡжңҖдҪҺдјҳе…Ҳзә§
    • 5пјҡNORM_PRIORITYпјҡй»ҳи®Өдјҳе…Ҳзә§
    • 10пјҡMAX_PRIORITYпјҡжңҖй«ҳдјҳе…Ҳзә§
  • зәҝзЁӢзҡ„дјҳе…Ҳзә§еҸӘжҳҜдёҖдёӘжҰӮзҺҮпјҢдјҳе…Ҳзә§й«ҳзҡ„зәҝзЁӢдёҚдёҖе®ҡдјҡе…Ҳжү§иЎҢ
  • setPriority(int newPriority)пјҡи®ҫзҪ®зәҝзЁӢзҡ„дјҳе…Ҳзә§
  • final int getPriority()пјҡиҝ”еӣһзәҝзЁӢзҡ„дјҳе…Ҳзә§

зӨәдҫӢпјҡ

3.3 setDaemon()

  • final void setDaemon(boolean on)пјҡе°ҶзәҝзЁӢи®ҫзҪ®дёәе®ҲжҠӨзәҝзЁӢ
  • еҪ“е…¶д»–зҡ„йқһе®ҲжҠӨзәҝзЁӢжү§иЎҢе®ҢжҜ•д№ӢеҗҺпјҢе®ҲжҠӨзәҝзЁӢдјҡйҷҶз»ӯз»“жқҹ

зӨәдҫӢпјҡ

3.4 yield()

  • public static void yield()пјҡеҮәи®©зәҝзЁӢ/зӨји®©зәҝзЁӢ
  • и®©еҪ“еүҚзәҝзЁӢеҮәи®©CPUзҡ„дҪҝз”ЁжқғпјҢеҪ“еүҚзәҝзЁӢдјҡиҝӣе…Ҙе°ұз»ӘзҠ¶жҖҒ

зӨәдҫӢпјҡ

3.5 join()

  • public static void join()пјҡжҸ’е…ҘзәҝзЁӢ/жҸ’йҳҹзәҝзЁӢ
  • е°ҶеҪ“еүҚзәҝзЁӢжҸ’е…ҘеҲ°жҢҮе®ҡзәҝзЁӢзҡ„еҗҺйқўпјҢзӯүеҫ…жҢҮе®ҡзәҝзЁӢжү§иЎҢе®ҢжҜ•д№ӢеҗҺпјҢеҪ“еүҚзәҝзЁӢжүҚдјҡз»§з»ӯжү§иЎҢ

зӨәдҫӢпјҡ

еӣӣгҖҒзәҝзЁӢзҡ„з”ҹе‘Ҫе‘Ёжңҹе’Ңе®үе…Ёй—®йўҳ

4.1 зәҝзЁӢзҡ„з”ҹе‘Ҫе‘Ёжңҹ

  • зәҝзЁӢзҡ„з”ҹе‘Ҫе‘Ёжңҹ
    • ж–°е»әзҠ¶жҖҒпјҡеҲӣе»әзәҝзЁӢеҜ№иұЎпјҢи°ғз”Ёstartж–№жі•д№ӢеүҚ
    • е°ұз»ӘзҠ¶жҖҒпјҡи°ғз”Ёstartж–№жі•д№ӢеҗҺпјҢзӯүеҫ…CPUеҲҶй…Қж—¶й—ҙзүҮ
    • иҝҗиЎҢзҠ¶жҖҒпјҡиҺ·еҫ—CPUзҡ„дҪҝз”ЁжқғпјҢжӯЈеңЁжү§иЎҢrunж–№жі•
    • йҳ»еЎһзҠ¶жҖҒпјҡи°ғз”ЁsleepгҖҒwaitгҖҒjoinзӯүж–№жі•пјҢзӯүеҫ…е…¶д»–зәҝзЁӢжү§иЎҢе®ҢжҜ•
    • жӯ»дәЎзҠ¶жҖҒпјҡrunж–№жі•жү§иЎҢе®ҢжҜ•пјҢзәҝзЁӢз»“жқҹ

4.2 зәҝзЁӢзҡ„е®үе…Ёй—®йўҳ

  • зәҝзЁӢзҡ„е®үе…Ёй—®йўҳпјҡеӨҡдёӘзәҝзЁӢеҗҢж—¶и®ҝй—®еҗҢдёҖдёӘиө„жәҗпјҢеҸҜиғҪдјҡеҜјиҮҙж•°жҚ®дёҚдёҖиҮҙзҡ„й—®йўҳ
  • и§ЈеҶіж–№ејҸпјҡдҪҝз”ЁеҗҢжӯҘд»Јз Ғеқ—

4.2.1 еҗҢжӯҘд»Јз Ғеқ—

  • еҗҢжӯҘд»Јз Ғеқ—ж јејҸ
  • зү№зӮ№
    • й”Ғй»ҳи®Өжү“ејҖпјҢжңүдёҖдёӘзәҝзЁӢиҝӣеҺ»дәҶпјҢй”ҒиҮӘеҠЁе…ій—ӯ
    • йҮҢйқўзҡ„д»Јз Ғе…ЁйғЁжү§иЎҢе®ҢжҜ•пјҢзәҝзЁӢеҮәжқҘпјҢй”ҒиҮӘеҠЁжү“ејҖ

зӨәдҫӢпјҡ

4.2.2 еҗҢжӯҘж–№жі•

  • еҗҢжӯҘж–№жі•пјҡе°ұжҳҜжҠҠsynchronizedе…ій”®еӯ—ж”ҫеңЁж–№жі•дёҠйқў
  • ж јејҸпјҡ
  • зү№зӮ№пјҡ
    • еҗҢжӯҘж–№жі•жҳҜй”ҒдҪҸж–№жі•йҮҢйқўзҡ„жүҖжңүд»Јз Ғ
    • й”ҒеҜ№иұЎдёҚиғҪиҮӘе·ұжҢҮе®ҡ
      • йқһйқҷжҖҒпјҡthis
      • йқҷжҖҒпјҡеҪ“еүҚзұ»зҡ„еӯ—иҠӮз Ғж–Үд»¶еҜ№иұЎMyThread.class

зӨәдҫӢпјҡ

4.3 Lockй”Ғ
