Leetcode 15.дёүж•°д№Ӣе’Ң

йўҳзӣ®иҰҒжұӮ

  • з»ҷдҪ дёҖдёӘж•ҙж•°ж•°з»„ nums пјҢеҲӨж–ӯжҳҜеҗҰеӯҳеңЁдёүе…ғз»„ [nums[i], nums[j], nums[k]] ж»Ўи¶і i != jгҖҒi != k дё” j != k пјҢеҗҢж—¶иҝҳж»Ўи¶і nums[i] + nums[j] + nums[k] == 0 гҖӮиҜ·дҪ иҝ”еӣһжүҖжңүе’Ңдёә 0 дё”дёҚйҮҚеӨҚзҡ„дёүе…ғз»„гҖӮ

  • жіЁж„Ҹпјҡзӯ”жЎҲдёӯдёҚеҸҜд»ҘеҢ…еҗ«йҮҚеӨҚзҡ„дёүе…ғз»„гҖӮ

зӨәдҫӢ 1пјҡ
иҫ“е…Ҙпјҡnums = [-1,0,1,2,-1,-4]
иҫ“еҮәпјҡ[[-1,-1,2],[-1,0,1]]
и§ЈйҮҠпјҡ
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 гҖӮ
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 гҖӮ
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 гҖӮ
дёҚеҗҢзҡ„дёүе…ғз»„жҳҜ [-1,0,1] е’Ң [-1,-1,2] гҖӮ
жіЁж„ҸпјҢиҫ“еҮәзҡ„йЎәеәҸе’Ңдёүе…ғз»„зҡ„йЎәеәҸе№¶дёҚйҮҚиҰҒгҖӮ

зӨәдҫӢ 2пјҡ
иҫ“е…Ҙпјҡnums = [0,1,1]
иҫ“еҮәпјҡ[]
и§ЈйҮҠпјҡе”ҜдёҖеҸҜиғҪзҡ„дёүе…ғз»„е’ҢдёҚдёә 0 гҖӮ

зӨәдҫӢ 3пјҡ
иҫ“е…Ҙпјҡnums = [0,0,0]
иҫ“еҮәпјҡ[[0,0,0]]
и§ЈйҮҠпјҡе”ҜдёҖеҸҜиғҪзҡ„дёүе…ғз»„е’Ңдёә 0 гҖӮ

жҸҗдәӨ

жҺ’еәҸ+еҸҢжҢҮй’Ҳ

йҰ–е…ҲеҜ№ж•°з»„иҝӣиЎҢжҺ’еәҸ
д№ӢеҗҺжҢҮй’Ҳiд»ҺеӨҙејҖе§ӢйҒҚеҺҶдҪңдёәaпјҢжҢҮй’Ҳleftе’ҢrightдҪңдёәbе’Ңc
еҰӮжһңsum=a+b+cеӨ§дәҺ0пјҢиҜҙжҳҺйңҖиҰҒsumеҸҳе°ҸпјҢжүҖд»ҘrightеҫҖе·Ұз§»пјҢеҰӮжһңе°ҸдәҺ0еҲҷиҜҙжҳҺsumйңҖиҰҒеҸҳеӨ§пјҢleftеҫҖеҸіз§»
йңҖиҰҒжіЁж„Ҹзҡ„жҳҜеҺ»йҮҚж“ҚдҪң

aзҡ„еҺ»йҮҚ

иҜҙеҲ°еҺ»йҮҚпјҢе…¶е®һдё»иҰҒиҖғиҷ‘дёүдёӘж•°зҡ„еҺ»йҮҚгҖӮ a, b ,c, еҜ№еә”зҡ„е°ұжҳҜ nums[i]пјҢnums[left]пјҢnums[right]

a еҰӮжһңйҮҚеӨҚдәҶжҖҺд№ҲеҠһпјҢaжҳҜnumsйҮҢйҒҚеҺҶзҡ„е…ғзҙ пјҢйӮЈд№Ҳеә”иҜҘзӣҙжҺҘи·іиҝҮеҺ»гҖӮ

дҪҶиҝҷйҮҢжңүдёҖдёӘй—®йўҳпјҢжҳҜеҲӨж–ӯ nums[i] дёҺ nums[i + 1]жҳҜеҗҰзӣёеҗҢпјҢиҝҳжҳҜеҲӨж–ӯ nums[i] дёҺ nums[i-1] жҳҜеҗҰзӣёеҗҢгҖӮ

жңүеҗҢеӯҰеҸҜиғҪжғіпјҢиҝҷдёҚйғҪдёҖж ·еҗ—гҖӮ

е…¶е®һдёҚдёҖж ·пјҒ

йғҪжҳҜе’Ң nums[i]иҝӣиЎҢжҜ”иҫғпјҢжҳҜжҜ”иҫғе®ғзҡ„еүҚдёҖдёӘпјҢиҝҳжҳҜжҜ”иҫғе®ғзҡ„еҗҺдёҖдёӘгҖӮ

еҰӮжһңжҲ‘д»¬зҡ„еҶҷжі•жҳҜ иҝҷж ·пјҡ

йӮЈжҲ‘д»¬е°ұжҠҠ дёүе…ғз»„дёӯеҮәзҺ°йҮҚеӨҚе…ғзҙ зҡ„жғ…еҶөзӣҙжҺҘpassжҺүдәҶгҖӮ дҫӢеҰӮ{-1, -1 ,2} иҝҷз»„ж•°жҚ®пјҢеҪ“йҒҚеҺҶеҲ°з¬¬дёҖдёӘ-1 зҡ„ж—¶еҖҷпјҢеҲӨж–ӯ дёӢдёҖдёӘд№ҹжҳҜ-1пјҢйӮЈиҝҷз»„ж•°жҚ®е°ұpassдәҶгҖӮ

жҲ‘д»¬иҰҒеҒҡзҡ„жҳҜ дёҚиғҪжңүйҮҚеӨҚзҡ„дёүе…ғз»„пјҢдҪҶдёүе…ғз»„еҶ…зҡ„е…ғзҙ жҳҜеҸҜд»ҘйҮҚеӨҚзҡ„пјҒ

жүҖд»ҘиҝҷйҮҢжҳҜжңүдёӨдёӘйҮҚеӨҚзҡ„з»ҙеәҰгҖӮ

йӮЈд№Ҳеә”иҜҘиҝҷд№ҲеҶҷпјҡ

иҝҷд№ҲеҶҷе°ұжҳҜеҪ“еүҚдҪҝз”Ё nums[i]пјҢжҲ‘д»¬еҲӨж–ӯеүҚдёҖдҪҚжҳҜдёҚжҳҜдёҖж ·зҡ„е…ғзҙ пјҢеңЁзңӢ {-1, -1 ,2} иҝҷз»„ж•°жҚ®пјҢеҪ“йҒҚеҺҶеҲ° з¬¬дёҖдёӘ -1 зҡ„ж—¶еҖҷпјҢеҸӘиҰҒеүҚдёҖдҪҚжІЎжңү-1пјҢйӮЈд№Ҳ {-1, -1 ,2} иҝҷз»„ж•°жҚ®дёҖж ·еҸҜд»Ҙж”¶еҪ•еҲ° з»“жһңйӣҶйҮҢгҖӮ

иҝҷжҳҜдёҖдёӘйқһеёёз»ҶиҠӮзҡ„жҖқиҖғиҝҮзЁӢгҖӮ

bдёҺcзҡ„еҺ»йҮҚ

еҫҲеӨҡеҗҢеӯҰеҶҷжң¬йўҳзҡ„ж—¶еҖҷпјҢеҺ»йҮҚзҡ„йҖ»иҫ‘еӨҡеҠ дәҶ еҜ№right е’Ңleft зҡ„еҺ»йҮҚпјҡпјҲд»Јз ҒдёӯжіЁйҮҠйғЁеҲҶпјү

дҪҶз»ҶжғідёҖдёӢпјҢиҝҷз§ҚеҺ»йҮҚе…¶е®һеҜ№жҸҗеҚҮзЁӢеәҸиҝҗиЎҢж•ҲзҺҮжҳҜжІЎжңүеё®еҠ©зҡ„гҖӮ

жӢҝrightеҺ»йҮҚдёәдҫӢпјҢеҚідҪҝдёҚеҠ иҝҷдёӘеҺ»йҮҚйҖ»иҫ‘пјҢдҫқз„¶ж №жҚ® while (right > left) е’Ң if (nums[i] + nums[left] + nums[right] > 0) еҺ»е®ҢжҲҗright-- зҡ„ж“ҚдҪңгҖӮ

жңҖзӣҙзҷҪзҡ„жҖқиҖғиҝҮзЁӢпјҢе°ұжҳҜrightиҝҳжҳҜдёҖдёӘж•°дёҖдёӘж•°зҡ„еҮҸдёӢеҺ»зҡ„пјҢжүҖд»ҘеңЁе“ӘйҮҢеҮҸзҡ„йғҪжҳҜдёҖж ·зҡ„гҖӮ
