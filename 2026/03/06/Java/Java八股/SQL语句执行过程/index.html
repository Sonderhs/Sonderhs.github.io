MySQLжһ¶жһ„

MySQLжһ¶жһ„еҰӮдёӢпјҡ

MySQLжһ¶жһ„е…ұеҲҶдёәдёӨеұӮпјҡ

  • ServerеұӮпјҡиҙҹиҙЈе»әз«ӢиҝһжҺҘгҖҒеҲҶжһҗе’Ңжү§иЎҢSQLгҖӮ
  • еӯҳеӮЁеј•ж“ҺеұӮпјҡеӨҚжқӮж•°жҚ®зҡ„еӯҳеӮЁе’ҢжҸҗеҸ–гҖӮ

MySQLзҡ„еҮ дёӘйҮҚиҰҒз»„д»¶пјҡ

  • иҝһжҺҘеҷЁпјҡиҙҹиҙЈеӨ„зҗҶе®ўжҲ·з«ҜиҝһжҺҘгҖӮ
  • жҹҘиҜўзј“еӯҳпјҡзј“еӯҳжҹҘиҜўз»“жһңд»ҘжҸҗй«ҳжҖ§иғҪгҖӮ
  • и§ЈжһҗеҷЁпјҡи§ЈжһҗSQLиҜӯеҸҘпјҢз”ҹжҲҗжҹҘиҜўи®ЎеҲ’гҖӮ
  • дјҳеҢ–еҷЁпјҡдјҳеҢ–жҹҘиҜўи®ЎеҲ’д»ҘжҸҗй«ҳжү§иЎҢж•ҲзҺҮгҖӮ
  • жү§иЎҢеҷЁпјҡжү§иЎҢжҹҘиҜўи®ЎеҲ’е№¶иҝ”еӣһз»“жһңгҖӮ

иҝһжҺҘеҷЁ

иҝһжҺҘеҷЁдё»иҰҒиҙҹиҙЈиҝһжҺҘMySQLжңҚеҠЎпјҢз„¶еҗҺжүҚиғҪжү§иЎҢSQLиҜӯеҸҘгҖӮ

иҝһжҺҘе‘Ҫд»Өпјҡ

MySQLжҳҜеҹәдәҺTCPеҚҸи®®зҡ„пјҢжүҖд»ҘиҝһжҺҘж—¶йңҖиҰҒз»ҸиҝҮTCPдёүж¬ЎжҸЎжүӢиҝҮзЁӢпјҢж–ӯејҖиҝһжҺҘж—¶йңҖиҰҒз»ҸиҝҮTCPеӣӣж¬ЎжҢҘжүӢиҝҮзЁӢгҖӮ

еёёз”ЁеҸӮж•°пјҡ

  • wait_timeoutпјҡз©әй—ІиҝһжҺҘжңҖеӨ§з©әй—Іж—¶й•ҝпјҢй»ҳи®ӨеҖјдёә8е°Ҹж—¶гҖӮ
  • max_connectionsпјҡжңҖеӨ§иҝһжҺҘж•°пјҢй»ҳи®ӨеҖјдёә151гҖӮ

MySQLиҝһжҺҘд№ҹеҲҶдёәй•ҝиҝһжҺҘе’ҢзҹӯиҝһжҺҘпјҢдҪҶй•ҝиҝһжҺҘжңүеҸҜиғҪдјҡеҚ з”ЁеҶ…еӯҳиҝҮеӨ§еҜјиҮҙMySQLжңҚеҠЎејӮеёёйҮҚеҗҜгҖӮ
и§ЈеҶіеҠһжі•пјҡ

  • е®ҡжңҹж–ӯејҖй•ҝиҝһжҺҘгҖӮ
  • е®ўжҲ·з«Ҝдё»еҠЁйҮҚзҪ®иҝһжҺҘгҖӮ

иҝһжҺҘеҷЁжҖ»з»“пјҡ

  • дёҺе®ўжҲ·з«ҜиҝӣиЎҢTCPдёүж¬ЎжҸЎжүӢе»әз«ӢиҝһжҺҘгҖӮ
  • ж ЎйӘҢе®ўжҲ·з«Ҝзҡ„з”ЁжҲ·еҗҚе’ҢеҜҶз ҒпјҢеҰӮжһңдёҚеҜ№еҲҷжҠҘй”ҷгҖӮ
  • еҰӮжһңеҜ№пјҢиҜ»еҸ–иҜҘз”ЁжҲ·жқғйҷҗпјҢеҗҺйқўзҡ„жқғйҷҗйҖ»иҫ‘еҲӨж–ӯйғҪеҹәдәҺжӯӨж—¶зҡ„жқғйҷҗгҖӮ

жҹҘиҜўзј“еӯҳ

жҹҘиҜўзј“еӯҳд»Ҙkey-valueзҡ„еҪўејҸеӯҳеӮЁжҹҘиҜўз»“жһңпјҢkeyжҳҜSQLжҹҘиҜўиҜӯеҸҘпјҢvalueжҳҜжҹҘиҜўз»“жһңгҖӮ

дҪҶеҜ№дәҺжӣҙж–°йў‘з№Ғзҡ„иЎЁпјҢжҹҘиҜўзј“еӯҳзҡ„е‘ҪдёӯзҺҮиҫғдҪҺпјҢеӣ дёәжҜҸж¬Ўжӣҙж–°йғҪдјҡеҜјиҮҙзӣёе…ізј“еӯҳиў«жё…з©әгҖӮ

жүҖд»ҘпјҢMySQL8.0зүҲжң¬е·Із»ҸеәҹејғдәҶжҹҘиҜўзј“еӯҳеҠҹиғҪгҖӮ8.0д№ӢеүҚзҡ„зүҲжң¬еҸҜд»ҘйҖҡиҝҮеҸӮж•°query_cache_typeи®ҫзҪ®жҲҗDEMANDжқҘе…ій—ӯжҹҘиҜўзј“еӯҳгҖӮ

и§ЈжһҗеҷЁ

и§ЈжһҗеҷЁиҙҹиҙЈе°ҶSQLиҜӯеҸҘи§ЈжһҗжҲҗMySQLиғҪеӨҹзҗҶи§Јзҡ„ж јејҸпјҢз”ҹжҲҗжҹҘиҜўи®ЎеҲ’гҖӮдё»иҰҒеҒҡдёӨд»¶дәӢпјҡ

  • иҜҚжі•еҲҶжһҗ
  • иҜӯжі•еҲҶжһҗ

иҜӯжі•еҲҶжһҗе°ҶиҜҚжі•еҲҶжһҗзҡ„з»“жһңж №жҚ®иҜӯжі•и§„еҲҷеҲӨж–ӯжҳҜеҗҰж»Ўи¶іMySQLиҜӯжі•пјҢжІЎй—®йўҳе°ұдјҡжһ„е»әеҮәSQLиҜӯжі•ж ‘пјҢж–№дҫҝеҗҺз»ӯиҺ·еҸ–SQLзұ»еһӢгҖҒиЎЁеҗҚгҖҒеӯ—ж®өеҗҚзӯүдҝЎжҒҜгҖӮ

жіЁж„Ҹпјҡи§ЈжһҗеҷЁеҸӘиҙҹиҙЈжЈҖжҹҘиҜӯжі•е’Ңжһ„е»әиҜӯжі•ж ‘пјҢдёҚдјҡеҺ»жҹҘиЎЁжҲ–иҖ…еӯ—ж®өеӯҳдёҚеӯҳеңЁгҖӮ

жү§иЎҢSQL

жү§иЎҢSQLдё»иҰҒеҲҶдёәдёүжӯҘпјҡ

  • prepareйҳ¶ж®өпјҡйў„еӨ„зҗҶйҳ¶ж®өгҖӮ
  • optimizeйҳ¶ж®өпјҡдјҳеҢ–йҳ¶ж®өгҖӮ
  • executeйҳ¶ж®өпјҡжү§иЎҢйҳ¶ж®өгҖӮ

йў„еӨ„зҗҶеҷЁ

prepareйҳ¶ж®өдё»иҰҒеҒҡдёӨд»¶дәӢпјҡ

  • жЈҖжҹҘSQLиҜӯеҸҘдёӯзҡ„иЎЁжҲ–иҖ…еӯ—ж®өжҳҜеҗҰеӯҳеңЁгҖӮ
  • е°Ҷselect *дёӯзҡ„*жү©еұ•дёәиЎЁдёҠзҡ„жүҖжңүеҲ—гҖӮ

дјҳеҢ–еҷЁ

дјҳеҢ–еҷЁдё»иҰҒиҙҹиҙЈзЎ®е®ҡSQLжҹҘиҜўиҜӯеҸҘжү§иЎҢж–№жЎҲпјҢжҜ”еҰӮйҖүжӢ©е“ӘдёӘзҙўеј•гҖҒдҪҝз”Ёд»Җд№ҲиҝһжҺҘж–№ејҸзӯүгҖӮ

еҸҜд»ҘйҖҡиҝҮexplainе‘Ҫд»ӨжҹҘзңӢSQLиҜӯеҸҘзҡ„жү§иЎҢи®ЎеҲ’гҖӮ

жү§иЎҢеҷЁ

жү§иЎҢеҷЁж №жҚ®дјҳеҢ–еҷЁз”ҹжҲҗзҡ„жү§иЎҢи®ЎеҲ’жқҘжү§иЎҢSQLиҜӯеҸҘпјҢжңҖз»Ҳиҝ”еӣһз»“жһңз»ҷе®ўжҲ·з«ҜгҖӮ

жҖ»з»“

жү§иЎҢдёҖжқЎSQLжҹҘиҜўиҜӯеҸҘзҡ„иҝҮзЁӢеҰӮдёӢпјҡ

  • иҝһжҺҘеҷЁпјҡе»әз«ӢиҝһжҺҘпјҢж ЎйӘҢз”ЁжҲ·еҗҚе’ҢеҜҶз ҒпјҢиҜ»еҸ–жқғйҷҗгҖӮ

  • жҹҘиҜўзј“еӯҳпјҡеҰӮжһңжҹҘиҜўзј“еӯҳе‘ҪдёӯпјҢеҲҷзӣҙжҺҘиҝ”еӣһз»“жһңпјҢеҗҰеҲҷз»§з»ӯжү§иЎҢгҖӮMySQL8.0зүҲжң¬е·Із»ҸеәҹејғдәҶжҹҘиҜўзј“еӯҳеҠҹиғҪгҖӮ

  • и§ЈжһҗеҷЁSQLпјҢйҖҡиҝҮи§ЈйҮҠеҷЁеҜ№SQLиҜӯеҸҘиҝӣиЎҢиҜҚжі•е’ҢиҜӯжі•еҲҶжһҗпјҢжһ„е»әиҜӯжі•ж ‘гҖӮ

  • жү§иЎҢSQLпјҢе…ұжңүдёүдёӘйҳ¶ж®өпјҡ

    • йў„еӨ„зҗҶйҳ¶ж®өпјҡжЈҖжҹҘиЎЁе’Ңеӯ—ж®өжҳҜеҗҰеӯҳеңЁпјҢжү©еұ•select *дёӯзҡ„*гҖӮ
    • дјҳеҢ–йҳ¶ж®өпјҡзЎ®е®ҡSQLжҹҘиҜўиҜӯеҸҘжү§иЎҢж–№жЎҲгҖӮ
    • жү§иЎҢйҳ¶ж®өпјҡж №жҚ®дјҳеҢ–еҷЁз”ҹжҲҗзҡ„жү§иЎҢи®ЎеҲ’жқҘжү§иЎҢSQLиҜӯеҸҘпјҢжңҖз»Ҳиҝ”еӣһз»“жһңз»ҷе®ўжҲ·з«ҜгҖӮ

  • жҹҘиҜўиҜӯеҸҘзҡ„жү§иЎҢжөҒзЁӢеҰӮдёӢпјҡжқғйҷҗж ЎйӘҢпјҲеҰӮжһңе‘Ҫдёӯзј“еӯҳпјүвҖ”>жҹҘиҜўзј“еӯҳвҖ”>еҲҶжһҗеҷЁвҖ”>дјҳеҢ–еҷЁвҖ”>жқғйҷҗж ЎйӘҢвҖ”>жү§иЎҢеҷЁвҖ”>еј•ж“Һ

  • жӣҙж–°иҜӯеҸҘжү§иЎҢжөҒзЁӢеҰӮдёӢпјҡеҲҶжһҗеҷЁ---->жқғйҷҗж ЎйӘҢ---->жү§иЎҢеҷЁвҖ”>еј•ж“ҺвҖ”redo log(prepare зҠ¶жҖҒ)вҖ”>binlogвҖ”>redo log(commit зҠ¶жҖҒ)
