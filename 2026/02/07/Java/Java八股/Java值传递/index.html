  • Java中只有值传递没有引用传递

1. 形参&实参

  • 形参：方法定义时的参数，形式参数
  • 实参：方法调用时的参数，实际参数
  • 示例：

2. 值传递&引用传递

  • 值传递：方法调用时传递进去的是实参的值，方法接受的是实参值的拷贝，而不是实参本身
  • 引用传递：方法调用时传递进去的是实参的地址，此时形参就是实参，对形参的修改都会同步到实参上
