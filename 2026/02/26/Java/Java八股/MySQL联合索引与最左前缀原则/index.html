иҒ”еҗҲзҙўеј•

иҒ”еҗҲзҙўеј•пјҡеңЁMySQLпјҲжӣҙеҮҶзЎ®иҜҙжҳҜInnoDBеј•ж“ҺпјүдёӯпјҢиҒ”еҗҲзҙўеј•е°ұжҳҜеҜ№еӨҡдёӘеҲ—дёҖиө·е»әз«ӢдёҖдёӘB+Treeзҙўеј•пјҢдҫӢеҰӮпјҡ

иҝҷдёҚжҳҜе»әдәҶдёүдёӘзҙўеј•пјҢиҖҢжҳҜе»әдәҶдёҖжЈөжҢүз…§(a,b,c)жҺ’еәҸзҡ„B+TreeгҖӮ

зҗҶи§Је…ій”®зӮ№пјҡ
иҝҷжЈөж ‘йҮҢзҡ„вҖңй”®вҖқжҳҜдёҖдёӘжңүеәҸзҡ„дёүе…ғз»„пјҡ

(a1, b1, c1)
(a1, b1, c2)
(a1, b2, c1)
(a2, b1, c1)

е®ғзҡ„жҺ’еәҸи§„еҲҷжҳҜпјҡ
е…ҲжҢү a жҺ’еәҸ вҶ’ a зӣёеҗҢжҢү b жҺ’еәҸ вҶ’ b зӣёеҗҢжҢү c жҺ’еәҸ

иҝҷе°ұеғҸеӯ—е…ёжҺ’еәҸпјҢе…ҲзңӢе§“пјҢеҶҚзңӢеҗҚпјҢеҶҚзңӢдёӯй—ҙеҗҚгҖӮ

жңҖе·ҰеүҚзјҖеҺҹеҲҷ

жңҖе·ҰеүҚзјҖеҺҹеҲҷпјҡжҹҘиҜўжқЎд»¶еҝ…йЎ»д»Һзҙўеј•зҡ„жңҖе·ҰеҲ—ејҖе§ӢпјҢдё”дёҚиғҪи·іиҝҮдёӯй—ҙеҲ—гҖӮ

еҰӮжһңдҪ дёҚжҢҮе®ҡ aпјҢзӣёеҪ“дәҺдҪ иҰҒеңЁдёҖжң¬вҖңжҢүе§“ж°ҸжҺ’еәҸзҡ„з”өиҜқз°ҝвҖқйҮҢжүҫжүҖжңүеҸ«вҖңеј дёүвҖқзҡ„дәәпјҢдҪҶдҪ дёҚзҹҘйҒ“е§“жҳҜд»Җд№ҲгҖӮйӮЈе°ұеҸӘиғҪе…Ёд№Ұзҝ»гҖӮ

дёәд»Җд№ҲвҖңдёҚиғҪи·іиҝҮеҲ—вҖқпјҹ

дҫӢеҰӮпјҡ

иҝҷйҮҢ a жҳҜжңҖе·ҰеҲ—пјҢжүҖд»ҘеҸҜд»Ҙз”Ёзҙўеј•е®ҡдҪҚеҲ° a=1 иҝҷж®өиҢғеӣҙгҖӮ
