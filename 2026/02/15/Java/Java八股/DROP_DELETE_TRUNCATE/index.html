|еӯ—ж•°жҖ»и®Ў:490|йҳ…иҜ»ж—¶й•ҝ:1еҲҶй’ҹ

DROPгҖҒDELETEдёҺTRUNCATE

  • DROPгҖҒDELETEе’ҢTRUNCATEйғҪжҳҜSQLдёӯз”ЁдәҺеҲ йҷӨж•°жҚ®зҡ„е‘Ҫд»ӨпјҢдҪҶе®ғд»¬зҡ„дҪңз”Ёе’ҢдҪҝз”ЁеңәжҷҜдёҚеҗҢгҖӮ

DROP

  • иҜӯжі•пјҡDROP TABLE иЎЁеҗҚ
  • дҪңз”Ёпјҡе®Ңе…ЁеҲ йҷӨж•ҙдёӘиЎЁпјҢеҢ…жӢ¬иЎЁз»“жһ„гҖҒж•°жҚ®гҖҒзҙўеј•гҖҒи§ҰеҸ‘еҷЁгҖҒзәҰжқҹзӯүжүҖжңүзӣёе…іеҜ№иұЎ
  • дҪҝз”ЁеңәжҷҜпјҡеҪ“иЎЁдёҚеҶҚйңҖиҰҒж—¶дҪҝз”Ё

DELETE

  • иҜӯжі•пјҡDELETE FROM иЎЁеҗҚ WHERE жқЎд»¶
  • дҪңз”ЁпјҡеҲ йҷӨиЎЁдёӯж»Ўи¶іжқЎд»¶зҡ„иЎҢпјҢдёҚеёҰWHEREеӯҗеҸҘдјҡеҲ йҷӨиЎЁдёӯжүҖжңүиЎҢпјҢдҪҶиЎЁз»“жһ„е’Ңзӣёе…іеҜ№иұЎд»Қз„¶еӯҳеңЁ
  • зү№зӮ№пјҡиҮӘеўһеӯ—ж®өпјҲAUTO_INCREMENTпјүдёҚдјҡйҮҚзҪ®пјҢз»§з»ӯд»Һд№ӢеүҚзҡ„еҖјйҖ’еўһ
  • дҪҝз”ЁеңәжҷҜпјҡеҪ“йңҖиҰҒеҲ йҷӨзү№е®ҡж•°жҚ®ж—¶дҪҝз”Ё

TRUNCATE

  • иҜӯжі•пјҡTRUNCATE TABLE иЎЁеҗҚ
  • дҪңз”ЁпјҡеҲ йҷӨиЎЁдёӯжүҖжңүиЎҢпјҢдҪҶдҝқз•ҷиЎЁз»“жһ„е’Ңзӣёе…іеҜ№иұЎ
  • зү№зӮ№пјҡиҮӘеўһеӯ—ж®өдјҡйҮҚзҪ®пјҢдёӢдёҖж¬ЎжҸ’е…Ҙдјҡд»Һ1ејҖе§Ӣ
  • дҪҝз”ЁеңәжҷҜпјҡеҪ“йңҖиҰҒеҝ«йҖҹжё…з©әиЎЁж•°жҚ®дҪҶдҝқз•ҷиЎЁз»“жһ„ж—¶дҪҝз”Ё

еҜ№иЎЁз»“жһ„зҡ„еҪұе“Қ

  • DROPпјҡеҲ йҷӨиЎЁз»“жһ„е’ҢжүҖжңүж•°жҚ®
  • DELETEпјҡдҝқз•ҷиЎЁз»“жһ„пјҢеҲ йҷӨж•°жҚ®
  • TRUNCATEпјҡдҝқз•ҷиЎЁз»“жһ„пјҢеҲ йҷӨж•°жҚ®

и§ҰеҸ‘еҷЁ

  • DELETEдјҡи§ҰеҸ‘DELETEи§ҰеҸ‘еҷЁ
  • TRUNCATEе’ҢDROPдёҚдјҡи§ҰеҸ‘DELETEи§ҰеҸ‘еҷЁ

дәӢеҠЎе’Ңеӣһж»ҡ

  • DROPпјҡеұһдәҺDDLж“ҚдҪңпјҢдёҚиғҪеӣһж»ҡ
  • DELETEпјҡеұһдәҺDMLж“ҚдҪңпјҢеҸҜд»Ҙеӣһж»ҡ
  • TRUNCATEпјҡеұһдәҺDDLж“ҚдҪңпјҢдёҚиғҪеӣһж»ҡ
  • жіЁпјҡ
    • DMLпјҡж•°жҚ®еә“ж“ҚдҪңиҜӯиЁҖпјҲData Manipulation LanguageпјүпјҢжҳҜжҢҮеҜ№ж•°жҚ®еә“дёӯиЎЁи®°еҪ•зҡ„ж“ҚдҪңпјҢдё»иҰҒеҢ…жӢ¬иЎЁи®°еҪ•зҡ„жҸ’е…ҘгҖҒжӣҙж–°гҖҒеҲ йҷӨе’ҢжҹҘиҜўпјҢжҳҜејҖеҸ‘дәәе‘ҳж—ҘеёёдҪҝз”ЁжңҖйў‘з№Ғзҡ„ж“ҚдҪңгҖӮ
    • DDLпјҡж•°жҚ®е®ҡд№үиҜӯиЁҖпјҲData Definition LanguageпјүпјҢжҳҜеҜ№ж•°жҚ®еә“еҶ…йғЁзҡ„еҜ№иұЎиҝӣиЎҢеҲӣе»әгҖҒеҲ йҷӨгҖҒдҝ®ж”№зҡ„ж“ҚдҪңиҜӯиЁҖгҖӮе®ғе’ҢDMLиҜӯиЁҖзҡ„жңҖеӨ§еҢәеҲ«жҳҜDMLеҸӘжҳҜеҜ№иЎЁеҶ…йғЁж•°жҚ®зҡ„ж“ҚдҪңпјҢиҖҢдёҚж¶үеҸҠеҲ°иЎЁзҡ„е®ҡд№үгҖҒз»“жһ„зҡ„дҝ®ж”№пјҢжӣҙдёҚдјҡж¶үеҸҠеҲ°е…¶д»–еҜ№иұЎгҖӮDDLиҜӯеҸҘжӣҙеӨҡзҡ„иў«ж•°жҚ®еә“з®ЎзҗҶе‘ҳпјҲDBAпјүжүҖдҪҝз”ЁпјҢдёҖиҲ¬зҡ„ејҖеҸ‘дәәе‘ҳеҫҲе°‘дҪҝз”ЁгҖӮ
