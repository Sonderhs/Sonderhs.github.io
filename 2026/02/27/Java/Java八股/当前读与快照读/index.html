еҪ“еүҚиҜ»дёҺеҝ«з…§иҜ»

еҝ«з…§иҜ»пјҲSnapshot Readпјү

еҝ«з…§иҜ»пјҡиҜ»еҸ–зҡ„жҳҜжҹҗдёӘж—¶й—ҙзӮ№зҡ„дёҖиҮҙжҖ§зүҲжң¬пјҢиҖҢдёҚжҳҜж•°жҚ®еә“жӯӨеҲ»зҡ„жңҖж–°еҖјгҖӮ

е®ғдҫқиө– MVCCпјҲеӨҡзүҲжң¬е№¶еҸ‘жҺ§еҲ¶пјүгҖӮжҜҸдёҖиЎҢж•°жҚ®еңЁ InnoDB йҮҢйғҪйҡҗеҗ«дёӨдёӘзүҲжң¬еҸ·пјҡеҲӣе»әзүҲжң¬е’ҢеҲ йҷӨзүҲжң¬гҖӮдәӢеҠЎејҖе§Ӣж—¶дјҡз”ҹжҲҗдёҖдёӘ Read ViewпјҲиҜ»и§ҶеӣҫпјүпјҢиҝҷдёӘи§ҶеӣҫеҶіе®ҡдәҶдҪ вҖңиғҪзңӢеҲ°е“ӘдәӣзүҲжң¬вҖқгҖӮ

е…ёеһӢзҡ„еҝ«з…§иҜ»иҜӯеҸҘпјҡ

еүҚжҸҗжҳҜжІЎжңүеҠ й”ҒгҖӮ

е®ғзҡ„зү№зӮ№жҳҜпјҡ

  • дёҚеҠ иЎҢй”Ғ
  • дёҚйҳ»еЎһеҶҷ
  • иҜ»зҡ„жҳҜвҖңеҺҶеҸІзүҲжң¬вҖқ

дҪ еҸҜд»ҘжҠҠе®ғзҗҶи§ЈжҲҗпјҡж•°жҚ®еә“з»ҷдҪ жӢҚдәҶдёҖеј вҖңж—¶й—ҙз…§зүҮвҖқгҖӮдҪ д№ӢеҗҺзҡ„жҷ®йҖҡжҹҘиҜўйғҪеңЁзңӢиҝҷеј з…§зүҮгҖӮ

дёҫдёӘдҫӢеӯҗгҖӮ

дәӢеҠЎAпјҡ

дәӢеҠЎBпјҡ

дәӢеҠЎAеҶҚж¬Ўпјҡ

еҰӮжһңжҳҜREPEATABLE READйҡ”зҰ»зә§еҲ«пјҢдәӢеҠЎAиҝҳжҳҜдјҡзңӢеҲ° 100гҖӮеӣ дёәе®ғеңЁз¬¬дёҖж¬ЎиҜ»ж—¶е·Із»ҸзЎ®е®ҡдәҶвҖңз…§зүҮж—¶й—ҙвҖқгҖӮ

е®ғиҜ»зҡ„жҳҜundo logйҮҢзҡ„ж—§зүҲжң¬пјҢиҖҢдёҚжҳҜж•°жҚ®йЎөдёҠзҡ„жңҖж–°еҖјгҖӮ

еҪ“еүҚиҜ»пјҲCurrent Readпјү

еҪ“еүҚиҜ»пјҡиҜ»еҸ–зҡ„жҳҜж•°жҚ®еә“еҪ“еүҚжңҖж–°зүҲжң¬пјҢе№¶дё”дјҡеҜ№иҜ»еҸ–зҡ„и®°еҪ•еҠ й”ҒгҖӮ

е…ёеһӢиҜӯеҸҘпјҡ

дҪ дјҡеҸ‘зҺ°дёҖдёӘи§„еҫӢпјҡеҮЎжҳҜвҖңеҸҜиғҪиҰҒж”№ж•°жҚ®вҖқзҡ„ж“ҚдҪңпјҢйғҪдјҡз”ЁеҪ“еүҚиҜ»гҖӮ

еӣ дёәдҪ иҰҒж”№ж•°жҚ®пјҢеҝ…йЎ»зЎ®и®ӨдҪ зңӢеҲ°зҡ„жҳҜвҖңзҺ°еңЁзңҹе®һеӯҳеңЁзҡ„йӮЈдёҖиЎҢвҖқпјҢеҗҰеҲҷе°ұдјҡеҮәзҺ°йҖ»иҫ‘й”ҷд№ұгҖӮ

жҜ”еҰӮпјҡ

дәӢеҠЎAпјҡ

иҝҷдёӘж“ҚдҪңдјҡпјҡ

1пјүиҜ»жңҖж–°зүҲжң¬
2пјүеҠ жҺ’д»–й”ҒпјҲXй”Ғпјү
3пјүйҳ»еЎһе…¶д»–дәӢеҠЎеҜ№иҝҷиЎҢзҡ„дҝ®ж”№

е®ғдёҚжҳҜзңӢеҺҶеҸІзүҲжң¬пјҢе®ғжҳҜејәиЎҢжӢүеҲ°вҖңзҺ°е®һдё–з•ҢвҖқгҖӮ

ж ёеҝғеҢәеҲ«еҜ№жҜ”

з”ЁдёҖеҸҘиҜқжҖ»з»“пјҡ

еҝ«з…§иҜ» = зңӢеҺҶеҸІзүҲжң¬пјҢдёҚеҠ й”Ғ
еҪ“еүҚиҜ» = зңӢжңҖж–°зүҲжң¬пјҢеҠ й”Ғ

еҶҚж·ұе…ҘдёҖзӮ№пјҡ

еҝ«з…§иҜ»йқ  MVCC е®һзҺ°дёҖиҮҙжҖ§
еҪ“еүҚиҜ»йқ й”Ғе®һзҺ°дёҖиҮҙжҖ§

еҝ«з…§иҜ»и§ЈеҶізҡ„жҳҜпјҡиҜ»-еҶҷе№¶еҸ‘й—®йўҳ
еҪ“еүҚиҜ»и§ЈеҶізҡ„жҳҜпјҡеҶҷ-еҶҷе№¶еҸ‘й—®йўҳ

дёәд»Җд№ҲйңҖиҰҒдёӨдёӘжңәеҲ¶пјҹ

иҝҷжҳҜдёӘеҫҲжңүж„ҸжҖқзҡ„и®ҫи®Ўй—®йўҳгҖӮ

еҰӮжһңжүҖжңүиҜ»йғҪеҠ й”ҒпјҢдјҡеҸ‘з”ҹд»Җд№Ҳпјҹ

  • еӨ§йҮҸиҜ»йҳ»еЎһеҶҷ
  • еҗһеҗҗйҮҸжҡҙи·Ң
  • ж•°жҚ®еә“йҖҖеҢ–жҲҗдёІиЎҢжү§иЎҢ

еҰӮжһңжүҖжңүиҜ»йғҪдёҚеҠ й”ҒпјҢдјҡеҸ‘з”ҹд»Җд№Ҳпјҹ

  • update ж—¶дёҚзҹҘйҒ“дҪ ж”№зҡ„жҳҜе“ӘдёӘзүҲжң¬
  • дёўеӨұжӣҙж–°
  • е№»иҜ»еӨұжҺ§

жүҖд»Ҙ InnoDB еҒҡдәҶдёҖдёӘйқһеёёиҒӘжҳҺзҡ„жҠҳдёӯпјҡ

  • жҷ®йҖҡжҹҘиҜўз”Ё MVCCпјҲд№җи§Ӯе№¶еҸ‘пјү
  • ж¶үеҸҠдҝ®ж”№зҡ„иҜ»з”ЁеҠ й”ҒпјҲжӮІи§Ӯе№¶еҸ‘пјү

е®ғдёҚжҳҜйқһй»‘еҚізҷҪпјҢиҖҢжҳҜвҖңжҢүеңәжҷҜеҲҮжҚўжӯҰеҷЁвҖқгҖӮ

е’Ңйҡ”зҰ»зә§еҲ«зҡ„е…ізі»

еңЁREPEATABLE READдёӢпјҡ

  • еҝ«з…§иҜ»дҪҝз”Ёз¬¬дёҖж¬Ўз”ҹжҲҗзҡ„Read View
  • еҪ“еүҚиҜ»жҜҸж¬ЎйғҪиҜ»жңҖж–°зүҲжң¬

иҝҷзӮ№йқһеёёе…ій”®гҖӮ

жүҖд»ҘдҪ еҸҜиғҪзңӢеҲ°дёҖдёӘвҖңзҺ°иұЎвҖқпјҡ

дәӢеҠЎйҮҢжҷ®йҖҡselectжҹҘеҲ° 100
дҪҶselect for updateжҹҘеҲ° 200

иҝҷдёҚжҳҜж•°жҚ®еә“зІҫзҘһеҲҶиЈӮгҖӮ
жҳҜдҪ з”Ёзҡ„жҳҜдёҚеҗҢзҡ„иҜ»жЁЎеһӢгҖӮ

дёҖдёӘе®№жҳ“иў«иҜҜи§Јзҡ„зӮ№

жңүдәәдјҡй—®пјҡ

ж—ўз„¶жңү MVCCпјҢдёәд»Җд№ҲиҝҳйңҖиҰҒ next-key lockпјҹ

еӣ дёә MVCC еҸӘиғҪи§ЈеҶівҖңиЎҢзүҲжң¬еҸҜи§ҒжҖ§вҖқпјҢ
дёҚиғҪи§ЈеҶівҖңиҢғеӣҙжҸ’е…ҘвҖқзҡ„й—®йўҳпјҲе№»иҜ»пјүгҖӮ

дёҫдёӘдҫӢеӯҗпјҡ

иҝҷжҳҜеҪ“еүҚиҜ»гҖӮ

е®ғдјҡеҠ  next-key lockпјҢй”ҒдҪҸеҢәй—ҙпјҢйҳІжӯўеҲ«дәәеҫҖиҝҷдёӘиҢғеӣҙжҸ’ж•°жҚ®гҖӮ
