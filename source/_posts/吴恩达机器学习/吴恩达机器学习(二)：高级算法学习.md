# 高级算法学习

# 一、神经网络的构建
**人工神经网络**（Artificial Neural Network，即ANN ），是20世纪80 年代以来人工智能领域兴起的研究热点。它从信息处理角度对人脑神经元网络进行抽象， 建立某种简单模型，按不同的连接方式组成不同的网络。在工程与学术界也常直接简称为神经网络或类神经网络。神经网络是一种运算模型，由大量的节点（或称神经元）之间相互联接构成。每个节点代表一种特定的输出函数，称为激励函数（activation function）。每两个节点间的连接都代表一个对于通过该连接信号的加权值，称之为权重，这相当于人工神经网络的记忆。网络的输出则依网络的连接方式，权重值和激励函数的不同而不同。而网络自身通常都是对自然界某种算法或者函数的逼近，也可能是对一种逻辑策略的表达。
## 1.1 需求预测
为了能更加直观地认识神经网络，我们可以从一个简单的需求预测案例开始。
加入我们现在已知一件衣服的价格，那么如何预测其是否畅销呢？
首先我们可以利用前面学到过的逻辑回归sigmoid函数进行拟合：
![](/image/吴恩达机器学习/吴恩达机器学习(二)/demand_prediction1.png)
我们的输入是衣服的价格，输出则是由sigmoid函数得到的畅销概率$f(x)=sigmoid(wx+b)$，我们将$f(x)$记为$a(activation)$，现在我们可以将这个单个的逻辑单元视为神经网络中的一个神经元，它的输入是衣服的价格$price$，输出则是改衣服成为畅销产品的概率$a$。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/demand_prediction2.png)
这就是神经网络中的一个最简单的神经元模型。

假如我们现在已知四种关于该衣服的参数，分别是：价格$price$、运费$shipping cost$、市场营销$marketing$、材料$material$。现在我们要利用这四个特征来对该种衣服的畅销度进行预测，那么我们首先可能会考虑到影响该件衣服畅销度的几个因素，比如：该衣服的负担能力$affordability$、衣服的知名度$awareness$以及该衣服的质量$perceived qualify$。其中，我们会认为影响负担能力的因素有价格和运费，影响知名度的因素有市场营销，而影响感知质量的因素有价格和衣服的材料。通过对这三个中间因素的分析，我们最后可以评测这件衣服是否能成为畅销品。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/demand_prediction3.png)
在神经网络中，我们将最初的四个特征划为一层，称之为输入层，最后的输出划为一层，称之为输出层，中间的三个特征我们称之为激活值，将他们三个为一层称之为隐藏层。在一个神经网络中，隐藏层可能只有一层，也可能有很多层。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/demand_prediction4.png)
当神经网络比较小的时候，我们可以很容易的找到输入层与隐藏层特征之间的关系，但是在大型神经网络中，由于输入特征有很多，我们无法一个个地选择隐藏层与输入层特征之间的关系，所以我们可以采用前面学到过的特征工程，使激活值可以访问输入的任何一个特征，从而实现神经网络的链接。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/demand_prediction5.png)

## 1.2 神经网络中的层
现在有一个神经网络，我们将向量$\vec x$作为输入值输入，在第一层神经网络中进行计算。在第一层中有三个神经元，这三个神经元每一个都可以实现一个小型的逻辑回归单元，每一个神经元都有两个参数：$\vec w$和$b$，它们的作用是输出一些激活值$a=g(\vec w \cdot \vec x +b)$，其中$g$表示sigmoid函数。为了方便我们将第一个神经元的参数以及输出的激活值表示为$\vec w_1,b_1,a_1$，以此类推，最终得到该层的输出值$\vec a$。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/neural_network_layer1.png)
之后我们将该层的输出作为输出层的输入输入到下一层，经过输出层的神经元计算最后得出最终输出$a=g(\vec w_1 \cdot \vec a +b_1)$，再根据该输出对衣服的畅销度进行预测。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/neural_network_layer2.png)
为了更好的区分每一层的输出，我们一般会在右上角加上中括号以及该层的层数，如第一层的三个神经元参数分别记为$\vec w^{[1]}_1,b^{[1]}_1$，$\vec w^{[1]}_2,b^{[1]}_2$,$\vec w^{[1]}_3,b^{[1]}_3$，第一层输出的激活值为$\vec a^{[1]}$。

如果我们将输入层作为第0层，那么我们可以得到一个通用的公式：第$l$层的第$j$个神经元输出$a^{[l]}_j=g(\vec w^{[l]}_j \cdot \vec a^{[l-1]} + b^{[l]}_j)$
![](/image/吴恩达机器学习/吴恩达机器学习(二)/notation.png)

if a2 >= 0.5:

## 1.4 前向传播
### 1.4.1 单层上的前向传播
在上节中我们分析了如何用代码实现神将网络的搭建，本节我们来具体分析一下实现的具体过程。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/forward_prop.png)
其中输入值为$\vec x$，第一层输出值为$\vec a^[1]$，第二层输出值为$\vec a^[2]$，代码内部实现如下：
```python
x = np.array([200, 17])

# 第一层内部计算
w1_1 = np.array([1, 2])
b1_1 = np.array([-1])
z1_1 = np.dot(w1_1,x)+b1_1
a1_1 = sigmoid(z1_1)

w1_2 = np.array([-3, 4])
b1_2 = np.array([1])
z1_2 = np.dot(w1_2,x)+b1_2
a1_2 = sigmoid(z1_2)

w1_3 = np.array([5， -6])
b1_3 = np.array([2])
z1_3 = np.dot(w1_3,x)+b1_3
a1_3 = sigmoid(z1_3)

# 最后组合得到向量a1
a1 = np.array([a1_1, a1_2, a1_3])

# 第二层内部计算
w2_1 = np.array([-7, 8, -9])
b2_1 = np.array([3])
z2_1 = np.dot(w2_1,a1)+b2_1
a2_1 = sigmoid(z1_3) 

# 最终得到输出为a2
a2 = a2_1
```

### 1.4.2 利用Numpy实现前向传播

![](/image/吴恩达机器学习/吴恩达机器学习(二)/forward_prop_in_numpy.png)

密集函数Dense的实现：
```python
def dense(a_in,W,b,g):
  units = W.shape[1] # W大写表示他是一个矩阵，W.shape[0]表示矩阵的行数，W.shape[1]表示矩阵的列数
  a_out = np.zeros(units) # 初始化一个包含units个元素的数组
  for j in range(units): 
    w=W[:,j] # W[:,j]表示从矩阵W中取出第j列赋值给w
    z=np.dot(w,a_in)+b[j]
    a_out[j] = g(z) # g函数为激活函数
  return a_out
```

顺序函数Sequential的实现：
```python
def sequential(x):
  a1 = dense(x, W1, b1)
  a2 = dense(a1, W2, b2)
  a3 = dense(a2, W3, b3)
  a4 = dense(a3, W4, b4)
  f_x = a4
  return f_x
```

## 2.1 模型训练
### 2.1.1 使用Tensorflow实现
在本节中我们使用一个新的案例：手写数字识别。我们需要做的是识别图中数字是0还是1。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/handwritten.png)
该模型一共有四层，分别为输入层，两个隐藏层以及一个输出层。第一个隐藏层包含25个神经元，第二个隐藏层包含15个神经元，输出层有1个神经元。

import tensorflow as tf

])

### 2.1.2 模型训练细节
#### 2.1.2.1 逻辑回归模型
在之前逻辑回归中，我们训练逻辑回归模型的步骤一共有三步：
1. 第一步：指定怎样在给定输入特征$\vec x$以及参数$\vec w$和$b$的情况下计算输出。
我们一般会使用sigmoid函数来构建模型，即
```python
z = np.dot(w, x)+b

f_x = 1/(1+np.exp(-z))
```
2. 第二步：指定损失函数和成本函数
损失函数用于评估单个训练示例，而成本函数$J(\vec{w},b) = {1\over m}\displaystyle\sum_{i=1}^m[L(f_{\vec w,b}(x^{(i)}),y^{(i)})]$则是在整个训练集上计算的所有示例的损失函数的平均值
```python
# logistic loss
loss = -y * np.log(f_x) - (1-y) * np.log(1-f_x)
```
3. 第三步：使用特定的算法对模型进行训练使得成本函数最小，以寻找最合适的$w$和$b$
在逻辑回归中，我们一般使用梯度下降算法来进行最小化成本函数
```python
w = w - alpha * dj_dw
b = b - alpha * dj_db
```

#### 2.1.2.2 神经网络模型
在Tensorflow中训练神经网络模型中的步骤与在逻辑回归中相似：
1. 第一步：指定怎样在给定输入特征$\vec x$以及参数$\vec w$和$b$的情况下计算输出，也就是使用前向传播算法来构建模型
![](/image/吴恩达机器学习/吴恩达机器学习(二)/training1.png)
```python
# 构建模型
model = Sequential([
    Dense(units=25, activation='sigmoid'),  # 第1层一共25个神经元，激活函数使用sigmoid函数
    Dense(units=25, activation='sigmoid'),  # 第2层一共15个神经元，激活函数使用sigmoid函数
    Dense(units=25, activation='sigmoid')  # 第3层一共1个神经元，激活函数使用sigmoid函数
])
```
2. 第二步：制定损失函数，也就是模型的编译
在神经网络训练中，我们一般使用二元交叉熵函数来作为损失函数
![](/image/吴恩达机器学习/吴恩达机器学习(二)/training2.png)
```python
# 模型编译
from tensorflow.keras.losses import BinaryCrossentropy
model.compile(loss=BinaryCrossentropy)  # 对模型进行编译，使用的损失函数为二元交叉熵函数
```
补充：上面问题是分类问题，所以使用二元交叉熵函数作为损失函数，如果我们处理的是回归问题的话，我们需要使用均方误差损失函数来作为损失函数
```python
from tensorflow.keras.losses import MeanSquaredError
model.compile(loss=MeanSquaredError)
```
3. 第三步：训练模型
在最小化成本函数时，我们依然可以使用梯度下降算法，但是在逻辑回归中，我们需要在代码中不断重复梯度下降的步骤：
$$\begin{align*} \text{repeat}\; \lbrace \newline
\;  w^{[l]}_j &= w^{[l]}_j -  \alpha \frac{\partial J(\vec{w},b)}{\partial w_j}   \; \newline 
 b^{[l]} &= b^{[l]} -  \alpha \frac{\partial J(\vec{w},b)}{\partial b}  \newline \rbrace
\end{align*}$$
而在tensorflow中，我们可以使用一种名为反向传播的算法来进行计算，在代码中我们只需要调用fit函数即可
![](/image/吴恩达机器学习/吴恩达机器学习(二)/training3.png)
```python
model.fit(X, Y, epochs=100)  # 对模型进行训练,epochs表示梯度下降的次数
```

## 2.2 激活函数
### 2.2.1 sigmoid激活函数的替代方案
在之前的神经元模型中，我们一直都使用sigmoid函数作为激活函数，但sigmoid函数只适用于二元分类问题，其结果只有0和1，所以我们需要了解一些其他函数以便于更好地构建模型。
比如在之前的判断衣服畅销度的问题中，我们把人们对于衣服的感知度分为0(不了解)和1(了解)，但实际上人们对于衣服的感知度是有程度区分的，所以我们可以使用同一个新的函数ReLU函数(Linear rectification function)作为激活函数。
ReLU函数的表达式为$g(z)=max(0,z)$，也就是当$z<$0时，输出结果为0，当$z>0$时，输出结果为$z$，这就可以解决感知度只能介于0和1之间的问题。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/relu.png)
除此之外，有些时候我们还会使用线性激活函数作为激活函数，其表达式为$g(z)=z$，也就相当于没有使用激活函数
![](/image/吴恩达机器学习/吴恩达机器学习(二)/functions.png)

### 2.2.2 如何选择合适的激活函数
我们现在已经知道了三个常用的激活函数：sigmoid函数、ReLU函数以及线性激活函数，那么我们该如何选择合适的激活函数来构建神经单元呢？
#### 2.2.2.1 输出层激活函数的选择
对于输出层我们一般分为两种情况：
![](/image/吴恩达机器学习/吴恩达机器学习(二)/output_layer.png)
1. 当我们在处理输出结果$y$为0或1的二元分类问题时，我们一般使用sigmoid函数作为激活函数，因为它能够很好地预测$y$等于0或1的概率。
2. 当我们在解决回归问题时，我们需要根据不同的问题来选择不同的激活函数：
   * 比如，当我们需要预测明天股票的走势时，输出结果有可能为正数，也有可能为负数，所以最好的选择就是使用现行激活函数作为激活函数
   * 假如我们需要预测房子的价格时，由于房子的价格只能为非负数，所以我们应该使用ReLU函数作为激活函数
   * 总之，我们需要根据预测结果来选择合适的激活函数

#### 2.2.2.2 隐藏层激活函数的选择
对于隐藏层，我们一般都会选择ReLU函数作为激活函数。
![](/image/吴恩达机器学习/吴恩达机器学习(二)/hidden_layer.png)
这是因为：
1. 相对于sigmoid函数来说，ReLU函数表达式更简单一些，所以在训练过程中会更快，效率会更高；
2. sigmoid函数左右两侧都比较平坦，这可能会导致在梯度下降的过程很慢，减慢学习速度。

所以在实际应用中，我们一般会在隐藏层选择ReLU函数作为激活函数，而在输出层选择sigmoid函数作为激活函数

## 2.3 为什么要使用激活函数
在神经网络的每一个神经元中，我们一般都会使用激活函数来对数据进行处理，那么我们为什么要使用激活函数呢？
假设我们在所有的神经元中都不使用激活函数，或者都是用线性激活函数，那么对于一个简单的神经网络，我们做如下分析：
![](/image/吴恩达机器学习/吴恩达机器学习(二)/linear.png)
输入值经过第一层后输出
$$a^{[1]}=w^{[1]}_1x+b^{[1]}_1$$
将$a^{[1]}$作为输出层的输入值输入后得到
$$\begin{align*}
  a^{[2]} &= w^{[2]}_1a^{[1]}+b^{[2]}_1 \\ 
  &= w^{[2]}_1(w^{[1]}_1x+b^{[1]}_1)+b^{[2]}_1 \\
  &= (w^{[2]}_1w^{[1]}_1)x+w^{[2]}_1b^{[1]}_1+b^{[2]}_1
\end{align*}$$
假如我们将$w^{[2]}_1w^{[1]}_1$看做$w$，将$w^{[2]}_1b^{[1]}_1+b^{[2]}_1$看做$b$，那么$a^{[2]}=wx+b$，即输出结果$a^{[2]}$与输入结果$x$仍然是呈线性关系，相当于我们模型中的隐藏层并未发挥实际作用，这也是为什么我们要选择使用激活函数，以及为什么在隐藏层中一般选择ReLU函数而不是选择线性激活函数作为激活函数的原因。
